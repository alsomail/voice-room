## 一、背景与目标

- **本 Task 的目标**：
  1. server 出栈双发（兼容期）：发 `Ping`/`Pong` 同时仍回应 `ping`/`pong`（兼容现有客户端 1 个版本）。

  4. Web（如使用 WS 心跳）同步升级（先验证是否使用，未使用则 N/A）。
  5. 测试 fixture 全量改 PascalCase。

### 修改详情

#### 1. 文件头注释更新（Line 6-9）
- **Before**: `{"type":"ping","msg_id":"uuid","payload":{},"timestamp":1713312000}`
- **After**: `{"type":"Ping","msg_id":"uuid","payload":{},"timestamp":1700000000000}`
- 对齐 `Ping.schema.json` 主协议格式

#### 2. `"ping"` handler — 新增 WARN deprecated 日志（TDS PING-2 要求）
```rust
tracing::warn!(
    msg_type = "ping",
    "DEPRECATED: received lowercase 'ping', use 'Ping' (PascalCase). Will be removed in next major version."
);
```

#### 3. `"ping"` handler — timestamp 修复（秒 → 毫秒）
- **Before**: `timestamp: chrono::Utc::now().timestamp()`
- **After**: `timestamp: chrono::Utc::now().timestamp_millis()`

#### 4. `"Ping"` handler — timestamp 修复（秒 → 毫秒）
- **Before**: `timestamp: chrono::Utc::now().timestamp()`

- **Before**: `let ts = chrono::Utc::now().timestamp();`
