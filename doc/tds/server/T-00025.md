# TDS: 创建房间 API 升级（封面/分类/密码/公告）(Task ID: T-00025)

## 一、背景与目标
- **关联需求**：E-10 创建房间完整化 ([phase1_room_governance §2.1.1](../../product/phase1_room_governance.md))
- **本 Task 的目标**：扩展 POST /api/v1/rooms 支持新字段；新增 PATCH /api/v1/rooms/:id 让房主改名/公告/分类；变更广播 RoomInfoUpdated。

## 二、方案设计

### 接口扩展
**POST /api/v1/rooms**（需 JWT）
```json
{
  "title":"中东夜话", "room_type":"normal",
  "cover_url":"/assets/covers/desert.png",  // 必须为白名单前缀
  "category":"chat",                         // chat|emotion|music|game|matchmaking|other
  "announcement":"欢迎来到中东夜话~",        // 可选，≤200 字符
  "password":"123456"                        // room_type=password 必填，6 位数字
}
```
校验：
| 字段 | 规则 | 错误码 |
|------|------|--------|
| cover_url | 以 `/assets/covers/` 开头 或 CDN 白名单；缺省为空串 | 40003 |
| category | 6 枚举值之一 | 40003 |
| announcement | ≤200 Unicode 字符 | 40003 |
| password | `^\d{6}$`，仅 room_type=password 时校验 | 40003 |

密码存储：`bcrypt(password)` 写 `password_hash`

**PATCH /api/v1/rooms/:id**（需 JWT，仅房主）
```json
{
  "title":"...",           // 可选
  "announcement":"...",    // 可选，空串表示清空
  "category":"music"       // 可选
}
```
- 至少一个字段，否则 40003
- 非房主 403（40301）
- 房间已 closed → 409（40901）
- 更新后广播 WS `RoomInfoUpdated`

### 新增 WS 广播 `RoomInfoUpdated`（S→C）
```json
{
  "type":"RoomInfoUpdated",
  "payload":{
    "room_id":"uuid","title":"...","announcement":"...","category":"music",
    "cover_url":"...","has_password":true
  },
  "timestamp": ...
}
```

### 封面白名单
```rust
const COVER_PREFIX_ALLOW: &[&str] = &["/assets/covers/", "https://cdn.voiceroom.example/covers/"];
fn validate_cover_url(url: &str) -> Result<()> {
    if url.is_empty() { return Ok(()); }
    if COVER_PREFIX_ALLOW.iter().any(|p| url.starts_with(p)) { Ok(()) }
    else { Err(Error::validation("invalid cover_url")) }
}
```
