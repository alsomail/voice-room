## 三、TDD 验收用例
- [ ] NobleRenewFailed 弹窗含"立即充值"按钮（跳 T-30060）
- [ ] NobleExpired in_grace_period=true 弹窗含"立即续费"
- [ ] NobleExpired in_grace_period=false 弹窗显示"贵族已过期"
- [ ] 同一 event_id 24h 内不重复弹（DataStore 校验 + ack POST）
- [ ] 通知权限缺失时静默（不崩溃）
- [ ] WorkManager 24h 前本地通知（基于 expire_at - now ≤ 24h 调度）
- [ ] NobleRenewSuccess 轻量 Toast 不打扰
