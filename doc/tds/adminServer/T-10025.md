## 三、TDD 验收用例
- [ ] 分页 P95 < 200ms（10w 行）
- [ ] 多条件过滤（user_id / state / provider / created_at 区间 / amount 区间）正确
- [ ] 跨租户隔离（admin 仅能看自身租户订单）
- [ ] 详情含 7 个状态时间戳 + `provider_response_raw` JSON
- [ ] 无 `payment.read` 权限 → 403
- [ ] 写 audit_log

### 🔴 协议路径绑定的强制验收
- [ ] **PROTO-1**：Web 端 dataProvider 调用 URL 字面量必须含 `/admin/payments/orders`
