# TDS: 送礼特效播放器 + 弹幕样式 (Task ID: T-30031)

### GiftEffectController 设计
```kotlin
class GiftEffectController @Inject constructor(
    private val wsBus: WebSocketEventBus,
    private val lottieCache: LottieCache
) {
    val fullscreenEffect: StateFlow<FullscreenAnim?> = MutableStateFlow(null)
    private val queue = Channel<GiftReceivedEvent>(Channel.BUFFERED)

    init {
        // 同时只播 1 个 L3，其他排队或丢弃（MVP 最多排 3 个）
        CoroutineScope(Dispatchers.Main).launch {
            wsBus.events.filterIsInstance<GiftReceivedEvent>().collect { evt ->
                if (evt.effectLevel >= 4) queue.trySend(evt)
            }
        }
        CoroutineScope(Dispatchers.Main).launch {
            for (evt in queue) {
                fullscreenEffect.value = FullscreenAnim(evt.animationUrl!!, evt.duration)
                delay(evt.duration)
                fullscreenEffect.value = null
            }
        }
    }
}
```

### L1 GiftChatMessage 样式
- 金色昵称（MenaTheme.Gold）
- 头像小图 16dp
- 礼物图标 24dp
- 文字："送给 {receiver} × {count}"
- 粗体（effect_level >= 3）
- 连击不新增弹幕，累加 count 更新同一 msg_id 的弹幕（用 msg_id 作 LazyColumn key 去重）

### L2 麦位光圈
- 在 MicSlot 组件外套 Modifier `.giftGlow(active=true, durationMs=2000)`
- 金色光圈 Scale 1.0→1.2→1.0 循环 2 次，然后 `active=false`
- 接收 GiftReceivedEvent 后根据 `receiver.user_id` 找对应 MicSlot 触发

### L3 全屏 Lottie
- 全屏 Box，半透明黑色背景(0.3)，中央 LottieAnimation
- 用户点击跳过 → 立即置 null
- Lottie JSON 由 animation_url 下载，首次从 CDN 拉（超时 3s），失败 fallback 本地动画

### 补偿消息（重连后）
- 重连时通过 `GetRoomSnapshot`（未定义？后续补）拿最近礼物记录；或由 Server 在 JoinRoom response 带 `recent_gifts[]`
- 补偿消息在 L1 弹幕里以普通样式驻列，**L2/L3 不回放**

### testTag
`Key('fullscreen_gift_overlay')`, `Key('gift_msg_{msgId}')`, `Key('btn_skip_fullscreen_gift')`

### 架构要点
- **防腐层**：Lottie SDK 隔离在 `ILottiePlayer` 接口后，业务层零 Lottie 依赖
- **testability**：`GiftEffectController(scope)` 构造参数注入，测试用 `backgroundScope` 消除 `UncompletedCoroutinesError`
- **L1/L2 同步**：`onGiftReceived()` 内同步更新 StateFlow，emit 后即可 assert，无需 advance
- **L3 队列**：`Channel(capacity=3)` + `for (evt in l3Queue)` 顺序播放，`skipFullscreen()` cancel delay job 实现跳过
