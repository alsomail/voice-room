## 需求描述

实现"我的"Tab 个人中心页面，展示用户信息、资产概览和功能入口列表。

## 交互逻辑

| 操作 | 行为 |
|------|------|
| 进入页面 | 从 `IUserRepository.getMe()` 拉取用户信息渲染 |
| 点击 ID 旁 📋 | 复制 ID 到剪贴板 + Toast "已复制" |
| 点击钻石余额 | Phase 1 占位：Toast "充值功能即将上线" |
| 点击编辑资料 | Phase 1 占位：Toast "编辑功能即将上线" |
| 点击设置 | 导航到 SettingsScreen（本期实现简易版：语言切换开关） |
| 点击关于我们 | 弹出 AlertDialog 显示 App 版本 + 简介 |
| 点击退出登录 | `AlertDialog` 二次确认 → 清除 JWT → 导航到 LoginScreen（清栈） |

## 异常流程

| 场景 | 处理 |
|------|------|
| 获取用户信息失败 | 显示本地缓存数据（Room DB） + 重试按钮 |
| 网络断开 | 使用本地缓存渲染 |

## 通用组件提取

使用 T-30018 中已定义的 `AvatarWithFrame`。

## TDD 验收标准

1. 进入个人中心后显示用户头像（`AvatarWithFrame` 金色边框）、昵称、ID
2. 钻石余额从 UserProfile 获取并显示
3. 点击 ID 旁复制按钮，剪贴板内容等于用户 ID
4. 退出登录二次确认后清除 JWT + 导航到 LoginScreen + 清除导航栈
5. `MenaTheme` 深色渐变背景
6. 网络异常时使用本地缓存数据渲染
