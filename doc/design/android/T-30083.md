# T-30083: Android 家族详情页

> **归属端**: Android
> **模块**: Family (E-11)
> **前置依赖**: T-00083 (家族详情 API), T-30082 (家族列表), T-30018 (MenaTheme)
> **覆盖 Tasks**: T-30083 (详情页) — 与 T-30084/85/86/87 子模块联动入口

## 入口

| 入口 | 交互 |
|------|------|
| 家族列表卡片点击 | 跳详情（来源 = list） |

## 角色 CTA 矩阵

| 角色 | Banner 主按钮 | 顶部菜单 ⋮ | 底部固定栏 |
|------|--------------|------------|------------|
| 未加入 | "+ 申请加入"（PENDING 时变 "申请审核中") | 分享 / 举报 | 隐藏 |
| 已加入 Member | "进入聊天"（替代申请按钮） | 分享 / 退出家族 / 举报 | "💬 进入家族聊天" |
| Admin | 同 Member | 同 Member + "审批申请"（带红点） | 同 Member |
| Owner | 同 Member | + "管理设置" + "解散家族" | 同 Member |
| 已加入其他家族 | "你已加入 XX 家族" 灰色禁用 | 分享 / 举报 | 隐藏 |

## 状态机

| 状态 | UI |
|------|---|
| `Loading` | Banner Skeleton + Tab Skeleton |
| `Idle (notMember)` | "申请加入"按钮可点 |
| `ApplicationPending` | 按钮变"审核中"（灰色）+ 倒计时 7 日 |
| `Idle (member)` | 按钮"进入聊天" + 底部固定栏出现 |
| `Idle (otherMember)` | 按钮"已加入其他家族"灰色 |
| `Submitting` | 按钮 loading |
| `Error` | Snackbar |

## 交互逻辑

| 操作 | 行为 |
|------|------|
| 点击 Banner 公告 | 全文 Dialog（Markdown 渲染，最长 2000 字） |
| 等级进度条接近升级 | 闪烁 + 提示"再贡献 X 即可升级" |
| 点击成员项 | BottomSheet（资料卡 + 角色操作，T-30085） |
| Tab 切换 | 状态保留，重复点击 Tab 滑回顶部 |
| Tab "贡献榜" | 嵌入 T-30087 部分组件（家族月榜 + 我的贡献条） |
| Tab "公告历史" | 倒序公告列表，Owner/Admin 显示"发布新公告"按钮 |
| 申请加入 | 弹起 BottomSheet 让用户填写"申请理由"（可选） |
| 申请审核中再点击 | Toast "您的申请正在审核中" |
| Owner 解散家族 | 二次确认（输入家族名称才允许）+ reason |
| 退出家族 | 二次确认 + 提示"30 天内不能加入新家族" |

## 异常流程

| 场景 | 处理 |
|------|------|
| 已被该家族拉黑 | 申请按钮置灰 + Tooltip "您不能申请该家族" |
| 已加入其他家族 | 按钮"已加入其他家族"灰色 + 跳自家族 |
| 家族已解散 | 全屏占位"该家族已解散" + 返回按钮 |
| 网络失败 | 顶部黄条 + 重试 |
| 等级即将升级广播 | Snackbar "🎉 家族升至 LV4！成员上限 +40" |

## ViewModel

```kotlin
data class FamilyDetailUiState(
    val family: FamilyVO? = null,
    val myMembership: FamilyMembership? = null,    // null=未加入；含 role / joinedAt
    val applicationState: ApplicationState = ApplicationState.None,  // None/Pending/Approved/Rejected
    val members: List<MemberVO> = emptyList(),
    val membersHasMore: Boolean = true,
    val announcements: List<AnnouncementVO> = emptyList(),
    val activeTab: Int = 0,
    val unreadChatCount: Int = 0,
    val phase: LoadPhase = LoadPhase.Loading,
    val errorMessage: String? = null,
    val pendingDissolveConfirm: Boolean = false
)

sealed class FamilyDetailEvent {
    object NavigateToChat : FamilyDetailEvent()
    object NavigateToBlocked : FamilyDetailEvent()    // 已被拉黑
    data class OpenMemberSheet(val member: MemberVO) : FamilyDetailEvent()
    data class ShowToast(val msg: String) : FamilyDetailEvent()
    object FamilyDissolved : FamilyDetailEvent()
    data class FamilyLevelUp(val newLevel: Int) : FamilyDetailEvent()
}
```

## 协议入口索引

| 操作 | 协议路径 |
|------|---------|
| 拉详情 | GET `/api/v1/families/:id` |
| 拉成员 | GET `/api/v1/families/:id/members?cursor=` |
| 申请加入 | POST `/api/v1/families/:id/applications` |
| 退出 | DELETE `/api/v1/families/:id/me` |
| 解散 | DELETE `/api/v1/families/:id` |
| 成员变更广播 | WS `FamilyMemberChanged` |
| 等级升级广播 | WS `FamilyLevelUp` |
| 公告变更 | WS `FamilyAnnouncementUpdated` |

## RTL 兼容

- Banner 进度条方向反向
- Tab 顺序保持顺读（成员/贡献榜/公告）
- 角色徽标位置 `start` 替代 `end`
