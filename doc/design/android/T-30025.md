# T-30025: 房间页视觉升级

> **归属端**: Android  
> **模块**: UI/Room  
> **前置依赖**: T-30018 (MenaTheme)  
> **已有基础**: T-30009 (RoomScreen) + T-30010 (RoomViewModel) + T-30011 (MicSlotCard) + T-30014 (ChatMessageList)  
> **参考**: [Android App 界面设计规范 §2.7](../../product/android_app_design.md)

---

## 需求描述

将房间页从默认白色主题升级为中东黑金风格。**核心 WS 逻辑、上麦/下麦、消息收发等功能不变，仅改造视觉层**。

## 变更清单

| 元素 | 现状 | 升级后 |
|------|------|--------|
| 背景 | 白色 | `MenaColors.Background` |
| TopBar | 基础白色 | 深色半透明 + 房间ID小字 + OnlineCountBadge |
| 主麦（1号麦） | 与其他麦位同尺寸 | 80dp + 金色光圈 + 居中突出 |
| 副麦（2-9号） | 60dp 三列 Grid | 60dp 四列排列 |
| 空麦位 | 灰色占位 | 虚线圆圈 + "+" 灰色图标 |
| 占位麦位 | 头像 | 头像 + 昵称白色小字 + 音浪动画（绿色） |
| 静音麦位 | 红色图标 | 头像上覆盖红色禁麦圆形 Badge |
| 系统消息 | 居中灰色 | 居中 `SystemMessage` 金黄色 |
| 用户消息 | 昵称+内容 | 昵称 `Primary` 金色 + 内容白色 |
| 弹幕区背景 | 白色 | 半透明深色 `Background.copy(alpha=0.8f)` |

## TDD 验收标准

1. 房间页使用 `MenaTheme`，深色背景
2. 主麦（1号）居中突出，80dp + 金色光圈
3. 副麦 60dp 四列排列，空麦位显示虚线+"+"
4. 系统消息金黄色居中，用户消息金色昵称+白色内容
5. **所有现有功能测试（上麦/下麦/消息收发/WS）不可回归**
6. RTL 布局正确，麦位排列不镜像（数字布局）
