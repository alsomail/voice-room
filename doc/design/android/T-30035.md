## 2. EventReportClient 组件设计

### 2.1 架构

```
┌────────────────────────────────────────────────┐
│ 业务代码 analytics.track("gift_send_success", p) │
└────────────────────────────────────────────────┘
                 ↓
┌────────────────────────────────────────────────┐
│ core/analytics/SentryAnalytics                 │
│  ├─ 合规检查：若 privacy_choice == "crash_only" │
│  │    非 Crash 事件 直接 drop                  │
│  ├─ 公共字段注入 (session_id, device_id, ...) │
│  └─ 脱敏过滤器 (手机号/JWT/精确地址)            │
└────────────────────────────────────────────────┘
                 ↓
┌────────────────────────────────────────────────┐
│ EventReportClient (Singleton)                  │
│  ├─ Room DAO: event_queue 表                   │
│  ├─ Throttler: 每 8 条 / 120s 触发 flush      │
│  └─ Dispatcher: 判断 WS 在线?                 │
│       ├─ 是 → WsClient.send(ReportEvent)     │
│       └─ 否 → RetrofitApi.postEventsBatch() │
└────────────────────────────────────────────────┘
                 ↓
┌─────────────┬────────────────┐
│ WS 通道     │ HTTP 通道      │
│ ReportEvent │ POST /events/batch │
└─────────────┴────────────────┘
```

### 2.2 Room 实体

```kotlin
@Entity(tableName = "event_queue")
data class EventEntity(
    @PrimaryKey(autoGenerate = true) val id: Long = 0,
    val eventName: String,
    val propertiesJson: String,  // JSON string
    val clientTs: Long,
    val sessionId: String,
    val retryCount: Int = 0,
    val createdAt: Long = System.currentTimeMillis()
)
```

### 2.3 节流策略
| 条件 | 行为 |
|------|------|
| 队列 ≥ 8 条 | 立即 flush |
| 距上次 flush ≥ 120s | 启动 flush |
| 应用切后台 | 立即 flush（WorkManager 托管） |
| 崩溃事件 | **绕过队列** 直接走 Sentry SDK |

### 2.4 容量保护
- 队列最大 1000 条：超出时 **drop 最旧未上报**
- 单条 `propertiesJson` ≥ 8KB 时截断并记录 `truncated=true`
- 失败重试：指数退避（2s / 8s / 30s / 2min），最多 10 次

## 4. 通用组件提取

| 组件 | 说明 |
|------|------|
| `PrivacyConsentDialog` | 独立 `@Composable`，可被其他合规场景复用 |
| `AnalyticsPort` | 接口层，业务代码唯一依赖 |
| `EventReportClient` | `object` / `@Singleton`，全局复用 |
| `AnalyticsLifecycleObserver` | 监听 `ProcessLifecycleOwner`，自动产生前后台事件 |
| `PiiFilter` | 脱敏工具类，输入 properties 返回过滤后的副本 |
