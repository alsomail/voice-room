## 二、方案设计

### 核心数据流

```
[RoomViewModel.sendMessage(content)]
           │
           ▼ PROTO-BINDING: wsClient.sendEnvelope SendMessage
  WsEnvelope.build(
    type = "SendMessage",
    payload = mapOf("content" to content),
    msgId  = UUID.randomUUID()
  )
  → JSON: {"type":"SendMessage","msg_id":"<uuid>","payload":{"content":"..."},"timestamp":<ms>}
           │
           ▼ IWebSocketClient.send(json)
  OkHttpWebSocketClient（生产）
  FakeWebSocketClient（测试）
           │
           ▼ WS /ws
  app/server/src/room/handler/chat.rs
    ::handle_send_message
  ① content 校验 1..=500 chars
  ② NOT_IN_ROOM / CHAT_MUTED 检查
  ③ msg_id 幂等去重
  ④ filter_content(content)
  ⑤ chat_repo.insert_message → DB id
  ⑥ broadcast_to_room(RoomMessage envelope)
  ⑦ ACK SendMessageResult{code:0}
```
