# TDS: ping/pong → Ping/Pong 三端同步升级（PascalCase 一致性）(Task ID: T-00108)

## 一、背景与目标

- **关联需求**：用户决策 #2 — `ping`/`pong` 与其他 28 信令 PascalCase 不一致，本 Task 全量升级。
- **本 Task 的目标**：
  1. server 出栈双发（兼容期）：发 `Ping`/`Pong` 同时仍回应 `ping`/`pong`（兼容现有客户端 1 个版本）。
  2. server 入栈：同时接受 `Ping`/`ping`（兼容期）；下个版本只接受 `Ping`。
  3. Android `OkHttpWebSocketClient` 心跳改发 `Ping`，期望收 `Pong`。
  4. Web（如使用 WS 心跳）同步升级（先验证是否使用，未使用则 N/A）。
  5. 测试 fixture 全量改 PascalCase。
