# T-30018: MenaTheme 中东黑金主题系统

> **归属端**: Android  
> **模块**: UI/Theme  
> **前置依赖**: 无  
> **参考**: [Android App 界面设计规范](../../product/android_app_design.md)

---

## 需求描述

封装中东黑金风格的 Material3 主题系统，作为全局视觉基础。所有后续 UI 任务依赖此主题。

## 设计规格

### 颜色令牌 (MenaColors)

| Token | 色值 | 用途 |
|-------|------|------|
| `Background` | `#1A1A2E` | 页面背景 |
| `Surface` | `#16213E` | 卡片/容器 |
| `SurfaceVariant` | `#0F3460` | 次级容器 |
| `Primary` | `#D4AF37` | 金色主色（按钮/强调） |
| `PrimaryBright` | `#FFD700` | 亮金色（VIP/高亮） |
| `OnBackground` | `#FFFFFF` | 主文字 |
| `OnBackgroundSecondary` | `#B0B0B0` | 次要文字 |
| `OnBackgroundTertiary` | `#6C6C6C` | 三级文字/未选中 |
| `Error` | `#E74C3C` | 错误/封禁 |
| `Success` | `#2ECC71` | 成功/在线 |
| `SystemMessage` | `#F39C12` | 系统消息 |

### 字体 (MenaTypography)

- 英文: Roboto / Google Sans
- 阿拉伯语: Noto Sans Arabic
- 标题: `titleLarge` (22sp bold)、`titleMedium` (16sp bold)
- 正文: `bodyMedium` (14sp)、`bodySmall` (12sp)
- 标签: `labelSmall` (11sp)

### 形状 (MenaShapes)

- 大卡片: `RoundedCornerShape(16.dp)`
- 按钮: `RoundedCornerShape(24.dp)` (胶囊)
- 输入框: `RoundedCornerShape(12.dp)`
- 头像: `CircleShape`

### RTL 全局配置

- `MenaTheme` Composable 内部使用 `CompositionLocalProvider(LocalLayoutDirection provides LayoutDirection.Rtl)` 包裹 content

## 通用组件提取

本任务需要输出的可复用组件：

| 组件 | 文件 | 说明 |
|------|------|------|
| `MenaTheme` | `core/theme/MenaTheme.kt` | 包裹 `MaterialTheme` + RTL + Colors + Typography |
| `GoldButton` | `core/theme/GoldButton.kt` | 金色渐变胶囊按钮 |
| `GoldOutlinedTextField` | `core/theme/GoldOutlinedTextField.kt` | 深色底+金色边框输入框 |
| `AvatarWithFrame` | `core/theme/AvatarWithFrame.kt` | 圆形头像 + 可选金色/VIP边框 |

## 自动化测试锚点

| testTag | 说明 | 断言 |
|---------|------|------|
| 无页面级 testTag | 主题系统为全局 Provider | 验证方式：在 Preview/Test 中检查 `MaterialTheme.colorScheme.background == MenaColors.Background` |

## TDD 验收标准

1. `MenaTheme {}` 内所有子组件自动使用黑金色系
2. `GoldButton` 渲染金色渐变背景 + 白色文字 + 24dp 圆角
3. `GoldOutlinedTextField` 渲染深色底 + 金色边框 + 白色文字
4. `AvatarWithFrame` 默认金色边框 2dp + 圆形裁剪
5. RTL 布局在 `MenaTheme` 内自动生效
