## 三、TDD 验收用例
- [ ] frame_url=null 时不渲染框（仅头像）
- [ ] frame_url 非空时使用 Coil 加载 + 缓存
- [ ] 等级越高框越炫（依赖 server 下发图片）
- [ ] RTL 不影响
- [ ] 加载失败 fallback 不渲染框（不阻塞头像显示）
