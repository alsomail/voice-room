# TDS: 协议字段级 AST 审计（升级 T-0000T）+ CI 阻塞合并 (Task ID: T-00106)

## 一、背景与目标

- **关联需求**：T-0000T 已上线但仅做**路径级**比对（client grep ↔ server grep ↔ TDS 绑定表），无法捕获**字段级失配**（如 server 写 `payload.mic_index`、Android 读顶层 `slotIndex`）。本 Task 升级到 AST 级 + JSON Schema 比对。
- **本 Task 的目标**：扩展 `scripts/audit/protocol-binding-audit.ts`：
  1. 解析 server `serde_json::json!(...)` 出栈宏 + Android `@SerializedName` + Web `z.object({...})`，提取真实字段名集合。
  2. 与 `doc/protocol/schemas/**/*.schema.json` 字段集合比对。
  3. 任何字段级 mismatch 报 P0、CI 阻塞 PR。
  4. 输出 Markdown + JSON 报告，含 file:line 双向锚点。
