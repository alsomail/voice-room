# TDS: 创建订单 + 唤起 Billing (Task ID: T-30062)

## 一、背景与目标
- **关联需求**：[phase1_payment_billing.md §6.2](../../product/phase1_payment_billing.md)。
- **目标**：用户点击 SKU → 调 `POST /payments/orders` 拿 order_id → `BillingPort.launch(sku, obfuscatedAccountId=order_id)` → 监听 onPurchasesUpdated。

## 二、方案设计
### 核心数据流
```
SKU 卡片点击 → debounce(1s) → RechargeViewModel::startPurchase(sku)
   ↓
RechargeRepository::createOrder(sku.sku_id) → POST /payments/orders
   ↓
{ order_id } → BillingPort.launchBillingFlow(sku, obfuscatedAccountId=order_id)
   ↓
purchasesFlow.collect → DomainPurchase(token=..., obfuscatedAccountId=order_id)
   ↓ 验证 obfuscatedAccountId == order_id（防御性）
   ↓ 转交 T-30063 verify+ack 模块（`PendingPurchaseQueue::enqueue`）
```
