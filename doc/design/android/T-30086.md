# T-30086: Android 家族聊天频道

> **归属端**: Android · **模块**: Family (E-11) · **前置依赖**: T-00086 (家族 IM 频道), T-30022 (IM Tab), T-30086 (本 Task)
> **覆盖 Tasks**: T-30086

## 1. 入口

| 来源 | 入口 |
|------|------|
| 家族详情底部固定栏 | "💬 进入家族聊天" |
| IM Tab → 家族子 Tab | 唯一入口（一人一家族）|
| 推送通知 | deep link `voiceroom://family/chat?id=` |

> **频道复用**：与服务端 WS 频道 `family:{family_id}` 一致，所有家族成员加入即自动订阅，退出家族即解除订阅。

## 3. 消息类型

| 类型 | 渲染 | 权限 |
|------|------|------|
| 普通文本 | 气泡 | 所有成员 |
| @mention | 高亮 + 推送给被 @ 者 | 所有 |
| 公告 (System Pinned) | 顶部固定卡片 | Owner/Admin 发 |
| 图片 | 缩略图 + 点击全屏 | 所有（≤5MB）|
| 语音 | 长按发送，最长 60s（复用 T-30021 录制） | 所有 |
| 系统通知 | 灰色横条："法蒂玛 已加入家族 / 哈立德 已退出 / 等级升至 LV4" | 自动 |
| 房间邀请卡片 | 卡片 + 一键进房 | 在房间的成员可发 |
| 红包 (Phase 2.5) | - | 延后 |

## 4. 输入与限流

- 频率：3 秒 1 条；爆刷自动禁言 5 分钟
- 字数：≤ 500 字符（aGB 计算）
- 链接：自动检测可疑链接 → 红字风险提示 + Owner 可一键移除
- @mention：唤起 BottomSheet 列出在线成员（最多 50 个）

## 5. 成员上下文行为

| 操作 | 行为 |
|------|------|
| 长按消息 | BottomSheet：复制 / 引用 / 举报 / 撤回（仅本人 2 分钟内 / Admin 可撤）|
| 点头像 | 弹 `UserProfileSheet`（T-30090）|
| 双击消息 | 表情回应（Phase 2.5）|
| 点击"加入房间"卡片 | 切回房间（如有当前房间二次确认）|
| 滑动到底部 | 标记已读（消除 IM Tab 红点）|

## 6. 异常流程

| 场景 | 处理 |
|------|------|
| 网络断开 | 顶部黄条"连接中…"+ 输入栏置灰 + 自动重连 |
| 消息发送失败 | 红色感叹号 + 长按重发 |
| 已被 Owner 禁言 | 输入栏黄条"您已被禁言至 21:30" + 输入禁用 |
| 退出家族后再访问 | 全屏占位"您不再是家族成员" |
| 家族解散 | 全屏占位 + 自动跳大厅 |
| 撤回超 2 分钟 | Toast "超过撤回时间" |
| 历史消息加载失败 | 重试按钮 |
| 多设备同步 | 已读位点上行（`MarkRead { msg_id }`）|

## 9. 协议入口索引

| 操作 | 协议 |
|------|------|
| 历史消息 | GET `/api/v1/families/:id/messages?cursor=&limit=50` |
| 发送 | WS `SendFamilyMessage { family_id, msg_id, type, content, mentions[] }` |
| 接收 | WS `FamilyMessage { ... }` |
| 撤回 | WS `RecallFamilyMessage { family_id, msg_id }` |
| 已读 | WS `MarkFamilyRead { family_id, msg_id }` |
| 禁言变更 | WS `FamilyMemberMuted { user_id, until }` |
| 公告变更 | WS `FamilyAnnouncementUpdated`（T-30083 复用）|

## 10. RTL / 性能 / 埋点

- 我方消息靠右（LTR 布局）→ RTL 模式下我方靠左，他方靠右
- 公告卡片中 emoji + RTL 文本混排使用 `Bidi` LRE/PDF 包裹防错位
- 列表 LazyColumn `items(messages, key = msg_id)` + 50 条窗口滑动加载
- 埋点：`family_chat_view`、`family_chat_send`（type/length/mention_count）、`family_chat_room_invite_click`、`family_chat_report`
