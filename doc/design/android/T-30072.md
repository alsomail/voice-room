# T-30072: Android 贵族进场特效

> **归属端**: Android · **模块**: Nobility (E-09) · **前置依赖**: T-00069 (NobleEntered 广播), T-30018 (MenaTheme), T-30005 (NobleAvatarFrame)
> **覆盖 Tasks**: T-30072

## 1. 触发场景

服务端在以下事件后**全员广播** WS `NobleEntered { user_id, level, room_id, nickname, avatar, badge_url }`：
1. 贵族用户 `JoinRoom` 成功
2. 贵族用户首次开通 / 升档（贵族中心购买成功后立即触发，房间内全员看到一次"晋升"特效）

## 2. 分档展现矩阵

| 档位 | 全屏特效 | 顶部跑马灯 | BGM | 闪光持续 | 适用 |
|------|---------|----------|-----|---------|------|
| LV1 骑士 | 无 | ✓（蓝色横幅 1.5s） | 无 | - | 仅小图标 |
| LV2 男爵 | 无 | ✓（紫色） | 无 | - | - |
| LV3 子爵 | 半屏粒子（左下飘入） | ✓ | 无 | 2s | - |
| LV4 伯爵 | 全屏车马仪仗 (Lottie) | ✓ | 6s 短 BGM | 4s | 用户可关闭 |
| LV5 公爵 | 全屏 + 黄金尘埃 + 镜头光晕 | ✓（金色，中文/阿语双语） | 8s BGM（古阿拉伯小号） | 6s | 用户可关闭 |
| LV6 国王 | 全屏 + 红毯 + 烟花 + 用户大头像中心定格 | ✓（红金，全员强制） | 12s 史诗 BGM | 8s | **不可关闭**（产品规则） |

## 3. 屏幕布局（LV5/LV6 全屏特效）

```
┌─────────────────────────────────────────┐
│ ╭───── 跑马灯：阿里 已驾临 ─────╮       │  ← 顶部 banner
│                                          │
│         ✨ Lottie 动画区 ✨              │
│      （金色光晕 + 用户头像 + 贵族框）    │
│                                          │
│              👑 阿里 公爵                 │
│                                          │
│          [👋 欢迎] [💎 送花] [✕ 跳过]   │  ← 互动 + 跳过
│                                          │
└─────────────────────────────────────────┘
```

**关键约束**：
- 全屏特效层级在房间 UI 之上、私聊弹窗之下；不阻塞 WS 接收
- 麦上用户语音不打断（特效播放期间 RTC 音频继续）
- 多个特效排队（FIFO）；同一用户 5 分钟内重复进场不再播放

## 4. 状态机

| 状态 | 说明 |
|------|------|
| `Idle` | 无特效 |
| `Queued` | 等待中（队列长度 > 0） |
| `Playing` | 正在播放 |
| `Skipped` | 用户点跳过（非 LV6） |
| `MutedBgm` | 用户 Setting 关闭 BGM |

## 5. 用户偏好（Settings）

> 入口：设置 → 房间 → 贵族特效

| 选项 | 默认 | 说明 |
|------|------|------|
| 关闭进场特效（LV1-5） | OFF | 关闭后只看跑马灯，不全屏阻塞 |
| 关闭 BGM | OFF | 关闭后只动画无声 |
| LV6 国王 | 强制开启 | 灰色不可改（产品规则） |

## 6. 异常流程

| 场景 | 处理 |
|------|------|
| Lottie 资源未下载 | 降级为静态贵族框 + 跑马灯 |
| BGM 文件缺失 | 静默播放动画 |
| 房主屏蔽该用户（拉黑） | 不展示特效，但跑马灯仍显示（公开信息） |
| 用户在送礼 / 点击图鉴等关键操作 | 特效以"叠加层"出现，不打断当前 BottomSheet |
| 队列堆积 ≥ 5（多贵族同时进场） | 仅首条全屏，其余降级为跑马灯 |
| 跨设备低端机型 (RAM < 2GB) | 自动降级为半屏粒子 + 跑马灯 |

## 7. 通用组件

- `NobleEntryOverlay`（新）：全屏特效容器，输入 `level / userInfo / onComplete / onSkip`
- `NobleEntryMarquee`（新）：顶部跑马灯
- `NobleEntryQueue`（新）：进场事件队列管理（IO 线程）
- `LottieAssetCache`（复用 T-30031）：动画资源缓存

## 10. 性能与降级

- 同时最多 1 个全屏特效在播；其余进入队列（最大 4，超出丢弃保留 marquee）
- LV6 强制 + 长 BGM 占用大：必须预热下载（应用启动后台预拉资源）
- Frame budget：60fps 设备保 60；30fps 设备 Lottie speed=0.6
- 资源 CDN 下载失败 → 降级动画 + 上报 Sentry

## 11. RTL

- 跑马灯阿语模式：从右向左滑入
- 中心动画镜像（金色光晕方向反转）
- "进入"图标朝向反转
