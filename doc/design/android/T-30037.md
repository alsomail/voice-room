> **模块**: UI/Room

## 需求描述

提供 8 张中东风预设封面供房主创建房间时选择，MVP 不开放自定义上传。

## 8 张预设清单（drawable resources）

| idx | 资源名 | 主题 |
|----|--------|------|
| 0 | `cover_desert.jpg` | 沙漠金辉 |
| 1 | `cover_mosque.jpg` | 清真寺剪影 |
| 2 | `cover_lantern.jpg` | 烛灯/摩洛哥灯 |
| 3 | `cover_falcon.jpg` | 猎鹰与黑金 |
| 4 | `cover_rose.jpg` | 阿拉伯玫瑰 |
| 5 | `cover_yacht.jpg` | 迪拜游艇 |
| 6 | `cover_sunset.jpg` | 沙海落日 |
| 7 | `cover_calligraphy.jpg` | 阿拉伯书法 |

所有封面须符合 MENA 审美（**禁止**人物/宗教敏感元素近景）。

## 交互

- 点卡片 → 设置为选中（金色 2dp 边框 + 对勾角标）
- 初始选中第 0 张
- "确定" → 关闭 BottomSheet 并通过回调返回 `cover_url`（预设 URL 形如 `preset://desert`）

## 通用组件

- `CoverOptionCard(coverId, selected)` — 单张卡片 Composable
- 父页面通过 `onCoverSelected: (String) -> Unit` 回调接收

## 测试锚点

| Key | 说明 |
|-----|------|
| `Key('cover_picker_sheet')` | 根容器 |
| `Key('cover_option_0')` ~ `Key('cover_option_7')` | 8 张卡片 |
| `Key('btn_confirm_cover')` | 确定按钮 |

## TDD 验收

1. 8 张卡片全部可见
2. 点击后选中态切换（仅一张可选中）
3. 默认第 0 张选中
4. 确定后父页封面缩略图更新
5. 任意卡片被点击时产生选中状态变化事件
