# TDS: 观众席 Bottom Sheet (Task ID: T-30039)

## 一、背景与目标
- **关联需求**：E-10 观众席查看 ([设计文档](../../design/android/T-30039.md))
- **本 Task 的目标**：`AudienceBottomSheet` 展示房间内所有成员（麦上置顶 + 观众），支持分页与实时更新。

## 二、方案设计

### 涉及文件
| 文件 | 类型 | 说明 |
|------|------|------|
| `feature/room/components/AudienceBottomSheet.kt` | 新增 |  |

| `data/api/RoomApi.kt` | 修改 | `listMembers(roomId, page, limit)` |
| `data/model/RoomMember.kt` | 新增 | DTO |
| `feature/room/components/MemberRow.kt` | 新增 | 单行 UI |

### State
```kotlin
data class AudienceUiState(
    val onMic: List<RoomMember> = emptyList(),
    val audience: List<RoomMember> = emptyList(),
    val total: Int = 0,
    val loading: Boolean = false,
    val currentPage: Int = 1,
    val hasMore: Boolean = true
)
```

### 布局
- `ModalBottomSheet` 70% 高度
- Header "观众席 ($total)"
- `LazyColumn`：
  - Section header "麦上 (N)" + onMic 列表
  - Section header "观众 (N)" + audience 列表
- 每行 `MemberRow`：头像 + 昵称 + 角色徽章（owner 👑 / admin 🛡️）+ 在线时长
- 滚动触发 loadMore

### WS 集成
- `UserJoined`：state 追加到 audience 尾
- `UserLeft`：移除对应 user
- `MicTaken`：从 audience 移入 onMic
- `MicLeft`：从 onMic 移入 audience
- `AdminChanged`：更新 role 字段

### 点击回调
`onMemberClick(member) -> openUserActionSheet(member)`（T-30040）

### testTag
`Key('audience_sheet')`, `Key('audience_item_${userId}')`, `Key('audience_header_on_mic')`, `Key('audience_header_observers')`

### 性能
- 100 人房间：LazyColumn + 稳定 key (`userId`) 保持滚动流畅
- 头像用 Coil 缓存
