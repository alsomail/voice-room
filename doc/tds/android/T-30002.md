# TDS: 登录 ViewModel (Task ID: T-30002)

## 一、背景与目标
- **关联需求**：product.md - 手机号一步登录
- **本 Task 的目标**：实现登录 ViewModel，通过 Retrofit 调用 App Server 登录接口，管理完整的登录状态流转（发送验证码、倒计时、登录、JWT 存储）。
- **前置依赖**：T-00003 (login 接口), T-30001

## 二、方案设计
### 核心数据流
```
LoginScreen → LoginViewModel.sendCode(phone) → AuthRepository → Retrofit POST /auth/send-code → 成功/失败状态
LoginScreen → LoginViewModel.login(phone, code) → AuthRepository → Retrofit POST /auth/login → JWT → DataStore 存储 → NavigateTo(Hall)
```

### 涉及文件清单
| 文件路径 | 变更类型 | 说明 |
|----------|----------|------|
| `app/android/app/src/main/java/com/voiceroom/feature/auth/LoginViewModel.kt` | 新增 | 登录 ViewModel |
| `app/android/app/src/main/java/com/voiceroom/data/repository/AuthRepository.kt` | 新增 | 认证仓库 |
| `app/android/app/src/main/java/com/voiceroom/data/remote/AuthApiService.kt` | 新增 | Retrofit 接口定义 |
| `app/android/app/src/main/java/com/voiceroom/data/local/TokenManager.kt` | 新增 | DataStore JWT 存储 |

### 接口 / 信令定义
- `POST /auth/send-code` — 请求发送验证码
- `POST /auth/login` — 手机号 + 验证码登录，返回 JWT

### ViewModel 状态设计
```kotlin
data class LoginUiState(
    val phone: String = "",
    val code: String = "",
    val isLoading: Boolean = false,
    val isSendingCode: Boolean = false,
    val cooldownSeconds: Int = 0,
    val error: String? = null,
    val isLoginSuccess: Boolean = false,
    val isNewUser: Boolean = false
)
```

## 三、TDD 验收用例
- [ ] 正向：sendCode 成功 → isSendingCode=false, cooldownSeconds=60
- [ ] 正向：login 成功 → token 存入 DataStore → isLoginSuccess=true
- [ ] 正向：新用户登录 → isNewUser=true（可展示新手引导）
- [ ] 正向：Loading 状态正确切换
- [ ] 异常：sendCode 失败 → error 展示错误信息
- [ ] 异常：验证码错误 → error="验证码错误"
- [ ] 异常：网络异常 → error 展示友好提示
