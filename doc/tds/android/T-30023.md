# TDS: 消息Tab占位页 (Task ID: T-30023)

## 一、背景与目标
- **关联需求**：IM 消息功能（product.md - 实时消息模块），当前 IM 尚未开发，需在消息 Tab 展示占位页告知用户"即将上线"。
- **本 Task 的目标**：创建通用 `PlaceholderScreen` Composable 组件，替换现有 `MessagesPlaceholder` 硬编码占位，实现可复用的"功能即将上线"占位页，供消息、个人中心等多 Tab 复用。

## 二、方案设计

### 核心数据流
```
MainScreen NavHost
  └─ composable(MainTab.MESSAGES.route)
       └─ PlaceholderScreen(
              icon = Icons.AutoMirrored.Outlined.Chat,
              title = "消息功能即将上线",
              subtitle = "敬请期待"
          )
```
纯 UI 组件，无网络请求、无 ViewModel、无状态管理。

### 设计决策

#### 1. PlaceholderScreen 组件设计
- **包路径**：`com.voice.room.android.core.ui.PlaceholderScreen`（放在 `core/ui/` 目录，属于通用 UI 组件，非 feature 层）
- **参数签名**：
  ```kotlin
  @Composable
  fun PlaceholderScreen(
      title: String,
      modifier: Modifier = Modifier,
      icon: ImageVector? = null,
      subtitle: String? = null,
  )
  ```
- **视觉规范**：
  - 背景：`MenaColors.Background`（深色 `#1A1A2E`，由外层 Scaffold 的 `containerColor` 提供，组件本身不设背景以避免重复绘制）
  - 图标：`64.dp`，颜色 `MenaColors.OnBackgroundTertiary`（灰色 `#6C6C6C`），可选
  - 标题：`MaterialTheme.typography.titleMedium`（16sp Bold），颜色 `MenaColors.OnBackgroundSecondary`（`#B0B0B0`）
  - 副标题：`MaterialTheme.typography.bodySmall`（12sp），颜色 `MenaColors.OnBackgroundTertiary`（`#6C6C6C`），可选
  - 布局：垂直居中 `Column(horizontalAlignment = CenterHorizontally)`，图标→8dp→标题→4dp→副标题
- **testTag**：`"placeholder_screen"`

#### 2. MessagesPlaceholder 改造
- 保留 `MessagesPlaceholder()` 函数签名（不破坏 MainScreen 调用点），内部委托给 `PlaceholderScreen`
- 传入参数：`icon = Icons.AutoMirrored.Outlined.Chat`、`title = "消息功能即将上线"`、`subtitle = "敬请期待"`
- 保留 `testTag("messages_placeholder")` 语义：外层包裹 `Box(modifier = Modifier.testTag("messages_placeholder"))` 确保现有 T-30020 测试不回归

#### 3. 复用示例（T-30024 可参考）
后续 `ProfilePlaceholder` 也可改造为：
```kotlin
PlaceholderScreen(
    icon = Icons.Outlined.Person,
    title = "个人中心即将上线",
)
```
