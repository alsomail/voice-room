### 页面结构（详见 [设计文档](../../design/android/T-30027.md)）
1. 顶栏：返回 + 标题"我的钱包"
2. 余额大卡片：💎 巨大金色数字 + "充值"按钮（Toast "即将上线"）
3. 流水列表 LazyColumn：收入(+绿)/支出(-红) + 图标 + 时间 + reason
4. 下拉刷新 `PullRefresh`
5. 空状态：插画 + "暂无流水"

### WS 事件订阅
- 通过 Hilt 注入共享的 `WebSocketEventBus`（或复用 T-30008 封装）
- 在 ViewModel `init`：
```kotlin
viewModelScope.launch {
    wsEventBus.events.filterIsInstance<BalanceUpdatedEvent>().collect {
        _state.update { it.copy(balance = event.newBalance) }
        // 同时刷新流水首页
        txnPager.refresh()
    }
}
```

### Retrofit API
```kotlin
interface WalletApi {
  @GET("/api/v1/wallet/balance")

                       @Query("type") type: String? = null): ApiResponse<PageDto<TxnDto>>
}
```
