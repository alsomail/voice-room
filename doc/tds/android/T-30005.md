# TDS: Android 大厅页 UI Compose (Task ID: T-30005)

## 一、背景与目标

- **关联需求**：product.md — 房间大厅（首页）功能，用户打开 App 后看到活跃房间列表并进入房间。
- **本 Task 的目标**：在 Android 端实现完整的大厅页数据链路（`GET /api/v1/rooms` → DTO → 领域模型 → ViewModel → Compose UI），以 `LazyVerticalGrid` 展示第一页房间列表，支持 Coil 异步加载房主头像、在线人数显示和点击进房回调。
- **前置依赖**：T-00008（App Server `GET /api/v1/rooms` 接口，已完成）。
- **后续依赖**：T-30006（Paging3 分页）、T-30007（创建房间 / 正式导航接入）。

### 接口定义（引用 protocol.md §3.2）

**领域层接口**

```kotlin
// domain/room/RoomItem.kt
data class RoomItem(
    val roomId: String,
    val title: String,
    val roomType: String,       // "normal" / "password" / "paid"
    val memberCount: Int,
    val maxMembers: Int,
    val ownerNickname: String,
    val ownerAvatar: String?,   // null 时 Coil 显示占位图
    val createdAt: String
)

// domain/room/IRoomRepository.kt
interface IRoomRepository {
    suspend fun getRooms(page: Int, size: Int): Result<RoomsPage>
}

data class RoomsPage(
    val total: Int,
    val page: Int,
    val items: List<RoomItem>
)
```

**数据层接口**

```kotlin
// data/remote/api/RoomApiService.kt
interface RoomApiService {
    @GET("rooms")
    suspend fun getRooms(
        @Query("page") page: Int,
        @Query("size") size: Int
    ): Response<ApiResponse<RoomListResponseData>>
}

// data/remote/model/RoomItemDto.kt
data class RoomItemDto(
    @SerializedName("room_id")        val roomId: String,
    @SerializedName("title")          val title: String,
    @SerializedName("room_type")      val roomType: String,
    @SerializedName("member_count")   val memberCount: Int,
    @SerializedName("max_members")    val maxMembers: Int,
    @SerializedName("owner_id")       val ownerId: String,
    @SerializedName("owner_nickname") val ownerNickname: String,
    @SerializedName("owner_avatar")   val ownerAvatar: String?,
    @SerializedName("created_at")     val createdAt: String
)

// data/remote/model/RoomListResponseData.kt
data class RoomListResponseData(
    @SerializedName("total") val total: Int,
    @SerializedName("page")  val page: Int,
    @SerializedName("size")  val size: Int,
    @SerializedName("items") val items: List<RoomItemDto>
)
```

**ViewModel / UI 状态**

```kotlin
// feature/room/HallUiState.kt
data class HallUiState(
    val rooms: List<RoomItem> = emptyList(),
    val isLoading: Boolean = false,
    val error: String? = null,
    val currentPage: Int = 1,
    val totalItems: Int = 0,
    val hasMore: Boolean = false       // = totalItems > currentPage * PAGE_SIZE
) {
    companion object {
        const val PAGE_SIZE = 20
    }
}
```
