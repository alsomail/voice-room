# T-30026: 房间底部操作栏升级

> **归属端**: Android  
> **模块**: UI/Room  
> **前置依赖**: T-30018 (MenaTheme), T-30025 (房间视觉升级)  
> **已有基础**: T-30015 (ChatInputBar) + T-30013 (上麦/下麦逻辑)  
> **参考**: [Android App 界面设计规范 §2.7](../../product/android_app_design.md)

---

## 需求描述

升级房间底部操作栏，从纯输入框扩展为包含功能按钮组的完整操作栏。

## 布局设计

```
┌──────────────────────────────────────────┐
│  [输入消息...]        🎤   🎁   ❤️   🚪   │
│   ↑ 点击弹出键盘     ↑    ↑    ↑    ↑    │
│   ChatInputField    Mic  Gift Emoji Exit │
│                     Toggle              │
└──────────────────────────────────────────┘
高度: 56dp, 背景: Surface (#16213E), 上边框: 1dp #0F3460
```

### 按钮规格

| 按钮 | 图标 | 颜色 | 状态 | 行为 |
|------|------|------|------|------|
| 输入框 | — | 深色底+灰色提示文字 | 始终可用 | 点击展开键盘输入 |
| 🎤 麦克风 | `Mic` / `MicOff` | 绿色(开)/红色(关) | 仅在麦上时可用 | 切换本地麦克风静音 |
| 🎁 礼物 | `CardGiftcard` | 灰色(禁用) | Phase 0.5 禁用 | Toast "礼物功能即将上线" |
| ❤️ 表情 | `EmojiEmotions` | 灰色(禁用) | Phase 0.5 禁用 | Toast "表情功能即将上线" |
| 🚪 退出 | `ExitToApp` | `Error` 红色 | 始终可用 | 退出房间，导航回大厅 |

### 麦克风状态机

```
不在麦上 → 🎤 按钮灰色禁用
在麦上 + 未静音 → 🎤 绿色 Mic 图标
在麦上 + 已静音 → 🎤 红色 MicOff 图标
```

## 交互逻辑

| 操作 | 行为 |
|------|------|
| 点击输入框 | 弹出软键盘，`imePadding()` 适配（已有） |
| 点击 🎤 | 切换 `isMuted` 状态，调用 `IMediaService.stopPublishAudio()` / `startPublishAudio()` |
| 点击 🎁 | Toast "礼物功能即将上线" |
| 点击 ❤️ | Toast "表情功能即将上线" |
| 点击 🚪 | `AlertDialog` 确认 → `leaveRoom()` → 导航回 MainScreen |

## TDD 验收标准

1. 底部操作栏显示输入框 + 4 个功能按钮
2. 麦克风按钮在不在麦上时灰色禁用
3. 麦克风按钮在麦上时可切换绿色/红色（静音/取消静音）
4. 礼物和表情按钮灰色禁用，点击 Toast 提示
5. 退出按钮二次确认后离开房间
6. **所有现有输入框/发送功能不可回归**
