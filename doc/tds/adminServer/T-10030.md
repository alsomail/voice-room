## 三、TDD 验收用例
- [ ] privileges JSON Schema 校验未通过 → 400 + code=40917 + 字段路径
- [ ] level UNIQUE 冲突 → 409
- [ ] 软删（is_active=false）后已持有用户保留贵族至 expire_at
- [ ] 上传文件类型不在白名单 → 400
- [ ] 上传文件超大 → 413
- [ ] 操作后 publish noble_tiers_invalidate；App Server 缓存 60s 内刷新
- [ ] 全部操作 audit_log 落库
- [ ] 无 `noble.write` 权限 → 403
