### 文档章节
1. **三环境切换命令矩阵**
   ```
   |              | local                              | staging                          | prod                                 |
   |--------------|------------------------------------|----------------------------------|--------------------------------------|
   | Server       | APP_PROFILE=dev cargo run          | （远端，无需本地起）             | （远端）                             |
   | AdminServer  | ADMIN_PROFILE=dev cargo run        | （远端）                         | （远端）                             |
   | Web          | npm --prefix app/web run dev       | npm ... run build -- --mode staging | （远端 CDN）                         |
   | Android      | ./gradlew installLocalDebug        | ./gradlew installStagingDebug    | （走 Play Store）                    |
   | E2E          | npm run e2e:local                  | npm run e2e:staging              | npm run e2e:prod-smoke               |
   ```
2. **Local 冷启动 5 步**
   1. `cp .env.example .env && cp tests/scripts/env/.env.local.example tests/scripts/env/.env.local`
   2. `docker compose up -d postgres redis`
   3. 各端起服务（`cargo run` × 2 + `vite`）
   4. `npm run preflight`（应全 ✅）
   5. `npm run e2e:local`
3. **故障排查表**
   - PG 端口被占 / Redis AUTH 失败 / Server 启动配置缺失 / Web 5173 占用 / Midscene 401 等
4. **CI 接入示例**：GitHub Actions workflow 片段
5. **staging/prod 凭据获取流程**：占位（团队补充实际渠道）
