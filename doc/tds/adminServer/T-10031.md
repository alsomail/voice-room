## 三、TDD 验收用例
- [ ] 仅 super_admin 通过；其他角色 403
- [ ] reason 必填
- [ ] grant 任意天数（1~365）
- [ ] revoke 立即生效（user_nobles 记录删除）
- [ ] 写 admin_logs + noble_history(event=admin_grant/admin_revoke)
- [ ] 触发 NobleChanged 单播给本人 + 若在房间则房间广播
- [ ] grant 不扣钻石；renew_channel='admin_grant'
- [ ] revoke 不退钻石（产品规则）

### 🔴 协议路径绑定
- [ ] **PROTO-1**：URL 严格匹配 `/admin/users/:id/noble/{grant|revoke}`
- [ ] **PROTO-2**：NobleChanged.payload.reason ∈ {admin_grant, admin_revoke}
