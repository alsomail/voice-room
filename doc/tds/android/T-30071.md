## 二、方案设计
### 核心数据流
```
NobleCenterScreen 选档 → "立即开通" 弹窗
  ├─ 钻石 tab：
  │    余额够 → POST /nobles/purchase { tier_id, msg_id, auto_renew, duration_days }
  │    余额不足 → 跳 T-30060 RechargeScreen
  └─ 真金 tab：
       POST /nobles/usd_purchase_intent { tier_id, duration_days, msg_id }
         ↓ 返回 { payment_order_id, sku_id, ... }
       T-30062 BillingPort.launch(sku_id, obfuscatedAccountId=payment_order_id)
         ↓ T-30063 verify+ack
       Server 端 verify 完成时 noble_grant(usd 通道) 触发 NobleChanged
   ↓
WS 收到 NobleChanged → 弹专属庆祝动画 → 跳回贵族中心刷新
```

## 三、TDD 验收用例
- [ ] 钻石不足跳 T-30060；返回后保留选择
- [ ] 真金完整走 E-08 verify+ack
