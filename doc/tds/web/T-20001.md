# TDS: 管理员登录页 UI (Task ID: T-20001)

## 一、背景与目标
- **关联需求**：product.md - Admin Web 管理后台
- **本 Task 的目标**：实现 Admin Web 的管理员登录页面，使用 Ant Design 组件构建登录表单，支持账号密码登录、记住密码、中英文切换，居中卡片布局带品牌 Logo。
- **前置依赖**：T-10002 (Admin 登录接口)

## 二、方案设计
### 核心数据流
用户输入账号密码 → LoginForm 组件 → onSubmit → Zustand useAuthStore.login() → apiClient.post('/admin/login') → Admin Server → 返回 JWT → 存入 localStorage → 跳转 /dashboard

### 涉及文件清单
| 文件路径 | 变更类型 | 说明 |
|----------|----------|------|
| `app/web/src/pages/login/index.tsx` | 新增 | 登录页面组件 |
| `app/web/src/pages/login/LoginForm.tsx` | 新增 | Ant Design Form 登录表单 |
| `app/web/src/pages/login/login.module.css` | 新增 | 登录页样式 |
| `app/web/src/core/network/apiClient.ts` | 修改 | 配置 Admin Server baseURL |

### 接口 / 信令定义
参见 `doc/protocol.md` §三 3.1 POST /api/v1/admin/login

### UI 设计要点
- Ant Design Form + Input + Button 组件
- 账号输入框：用户名（非手机号）
- 密码输入框：Input.Password
- "记住密码" Checkbox（localStorage 存储 username）
- 登录失败展示 message.error() 提示
- 支持中英文（i18n，通过 useTranslation）
- 居中卡片布局，品牌 Logo 在上方

## 三、TDD 验收用例
- [ ] 正向：输入正确账号密码，点击登录，调用 API，成功跳转 /dashboard
- [ ] 正向：勾选"记住密码"，刷新后用户名自动填入
- [ ] 正向：登录失败展示错误提示（账号错误 / 密码错误 / 账号被禁用）
- [ ] 正向：支持中英文切换
- [ ] 异常：空表单提交显示表单校验错误
- [ ] 异常：网络异常显示友好提示
