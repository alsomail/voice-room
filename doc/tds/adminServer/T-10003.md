# TDS: 管理员 JWT 中间件 + RBAC (Task ID: T-10003)

## 一、背景与目标
- **关联需求**：所有需要认证的 Admin 接口
- **前置依赖**：T-10002 (管理员登录接口)
- **本 Task 的目标**：实现 Axum 中间件，校验 Admin JWT 并基于 role 进行 RBAC 权限控制

## 二、方案设计
### 核心数据流
Request → Authorization Header → JWT 中间件 → shared::jwt::decode_jwt() → 验证 iss="voiceroom-admin" → 提取 {admin_id, role} → RBAC 中间件 → 检查 role 是否有权访问该 endpoint → 注入 AdminAuthContext → next handler

### 涉及文件清单
| 文件路径 | 变更类型 | 说明 |
|----------|----------|------|
| `app/adminServer/src/common/middleware/jwt_auth.rs` | 新增 | Admin JWT 校验中间件 |
| `app/adminServer/src/common/middleware/rbac.rs` | 新增 | RBAC 权限校验中间件 |
| `app/adminServer/src/common/middleware/audit.rs` | 新增 | 审计日志自动记录中间件 |
| `app/adminServer/src/common/auth/context.rs` | 新增 | AdminAuthContext { admin_id, role } |
| `app/adminServer/src/common/auth/claims.rs` | 新增 | AdminClaims { sub, role, iss } |
