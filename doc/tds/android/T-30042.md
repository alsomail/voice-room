### UserKicked 流程
```
WS UserKicked 到达
   ↓
RoomViewModel.onKicked(reason, cooldown_sec)
   ↓
显示全屏 UserKickedDialog
   ├ 文案 "你已被移出房间，原因：XXX，10 分钟后可再次进入"
   ├ [知道了] 按钮
   ↓ 点击后
KickCooldownStore.save(roomId, until = now + 600s)
离开 RoomScreen，返回 Hall
   ↓ Hall 点击同一房间卡片前
if (KickCooldownStore.get(roomId).isFuture) {
    Toast("你暂时不能进入此房间，还剩 N 秒")
    return
}
```

### UserMuted 流程
```kotlin
// RoomViewModel 收到 UserMuted 且 target_user_id == self
when (type) {
    "mic"  -> muteViewModel.startMicCountdown(expiresAt)
    "chat" -> muteViewModel.startChatCountdown(expiresAt)
}
// duration_sec=0 则解除
```

`MuteStatusChip` 在房间顶部显示 "你已被禁{麦|言} N:NN" 每秒刷新，到期自动消失。
