# T-30021: 登录页视觉升级

> **归属端**: Android  
> **模块**: UI/Auth  
> **前置依赖**: T-30018 (MenaTheme)  
> **已有基础**: T-30001 (LoginScreen 功能完整)  
> **参考**: [Android App 界面设计规范 §2.2](../../product/android_app_design.md)

---

## 需求描述

将现有登录页从默认 Material3 白色主题升级为中东黑金视觉风格。**功能逻辑不变，仅改造视觉层**。

## 页面布局（升级后）

```
┌──────────────────────────────┐
│  (渐变背景 #1A1A2E → #16213E) │
│                              │
│         [App Logo]           │  ← 金色，80dp，居中
│                              │
│  ┌────────────────────────┐  │
│  │ +966 | 手机号输入框     │  │  ← GoldOutlinedTextField
│  └────────────────────────┘  │     深色底+金色边框
│                              │
│  ┌──────────────┐  ┌──────┐  │
│  │ 验证码输入框   │  │获取码│  │  ← 获取码=GoldButton小尺寸
│  └──────────────┘  └──────┘  │
│                              │
│  ┌────────────────────────┐  │
│  │       登  录            │  │  ← GoldButton 全宽，金色渐变
│  └────────────────────────┘  │
│                              │
│  登录即同意《用户协议》        │  ← 灰色 bodySmall
└──────────────────────────────┘
```

## 变更清单

| 元素 | 现状 | 升级后 |
|------|------|--------|
| 背景 | 白色 | 渐变深色 `#1A1A2E → #16213E` |
| Logo | 无 | 新增金色 App Logo 80dp |
| 手机号输入框 | Material3 默认 OutlinedTextField | `GoldOutlinedTextField`（深色底+金色边框） |
| 验证码输入框 | 同上 | `GoldOutlinedTextField` |
| 获取验证码按钮 | Material3 默认按钮 | `GoldButton` 小尺寸 |
| 登录按钮 | Material3 默认 FilledButton | `GoldButton` 全宽金色渐变 |
| 文字颜色 | 黑色 | 白色（主）/灰色（次） |

## 不变项

- ✅ +966 区号选择逻辑
- ✅ 60s 倒计时
- ✅ LoginViewModel / LoginUiState 状态管理
- ✅ RTL 布局支持
- ✅ 所有现有 testTag

## TDD 验收标准

1. 登录页使用 `MenaTheme` 包裹，深色渐变背景
2. 所有输入框使用 `GoldOutlinedTextField`
3. 所有按钮使用 `GoldButton`
4. Logo 金色显示在顶部
5. **所有现有功能测试用例不可回归**（登录/验证码/倒计时/RTL）
