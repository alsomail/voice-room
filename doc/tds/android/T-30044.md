# TDS: 禁麦/禁言 UI 反馈 + 抱麦集成 (Task ID: T-30044)

## 一、背景与目标
- **关联需求**：E-10 本地 UI 对禁/抱的硬约束 ([设计文档](../../design/android/T-30044.md))

### State
```kotlin
data class SelfGovernanceState(
    val micMutedUntil: Instant? = null,
    val chatMutedUntil: Instant? = null
) {
    val micMuted: Boolean get() = micMutedUntil?.isAfter(now()) == true
    val chatMuted: Boolean get() = chatMutedUntil?.isAfter(now()) == true
}
```

### ChatInput
```kotlin
OutlinedTextField(
    enabled = !state.chatMuted,
    placeholder = {
        Text(if (state.chatMuted)
            stringResource(R.string.chat_muted_placeholder, remainingMin)
            else stringResource(R.string.chat_placeholder))
    }
)
// 发送按钮同步置灰
```

### MicSlot "+" 按钮
```kotlin
IconButton(
    onClick = {
        if (state.micMuted) {
            Toast("你已被禁麦，暂不能上麦")
            return@IconButton
        }
        viewModel.takeMic(slotIndex)
    },
    enabled = !state.micMuted    // 视觉灰
)
```

### 被 ForceTakeMic 流程
```kotlin
WsEvent.MicTaken (target_user_id=self, forced_by != null) ->
    if (!permissionManager.hasMic()) {
        permissionManager.request().onResult { granted ->
            if (granted) rtcEngine.startPublishing(slot)
            else wsClient.send(MicLeave(slot))   // 自动下麦
        }
    } else {
        rtcEngine.startPublishing(slot)
    }
```

### 被 ForceLeaveMic 流程
```kotlin
WsEvent.MicLeft (target_user_id=self, forced_by != null) ->
    rtcEngine.stopPublishing()
    Toast("你已被抱下麦")
    // UI 状态由 roomState.onMicSelf = false 驱动
```

### testTag
`Key('chat_input')` + `enabled/disabled` 属性断言、`Key('mic_slot_$idx')`
