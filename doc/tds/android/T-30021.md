# TDS: 登录页视觉升级 (Task ID: T-30021)

## 一、背景与目标
- **关联需求**：Phase 0.5 - 交互壳体与基础体验（中东黑金主题统一）
- **前置依赖**：T-30018（MenaTheme 中东黑金主题系统 ✅ Done）
- **本 Task 的目标**：将现有 LoginScreen 从默认 Material3 白色主题改造为中东黑金风格（深色渐变背景 + GoldOutlinedTextField + GoldButton），功能逻辑完全不变，仅做视觉层改造。

## 二、方案设计

### 核心数据流

本任务为**纯 UI 视觉改造**，不涉及数据流变更。LoginViewModel / LoginUiState / IAuthRepository 均不修改。

```
改造前：LoginScreen → Surface(MaterialTheme.colorScheme.background) → OutlinedTextField / Button
改造后：LoginScreen → Box(MenaColors 渐变背景) → GoldOutlinedTextField / GoldButton
                       ↑                              ↑
                  MenaTheme 包裹                  core/theme 组件
```

### 设计决策

1. **渐变背景实现方式**：使用 `Brush.verticalGradient(MenaColors.Background → MenaColors.Surface)` 替代 `Surface(color=...)`，产生从深蓝黑到深藏蓝的纵向渐变，视觉层次更丰富。
2. **子组件改造策略**：`PhoneInput` / `CodeInput` / `CountdownButton` 三个子组件内部的 `OutlinedTextField` 和 `Button` 需替换为 `GoldOutlinedTextField` 和 `GoldButton`，以保持整页视觉一致。
3. **MenaTheme 包裹**：Preview 中将 `MaterialTheme {}` 替换为 `MenaTheme {}`，确保 Preview 展示黑金效果。
4. **文字颜色**：标题和副标题使用 `MenaColors.OnBackground`（白色）和 `MenaColors.Primary`（金色），替代默认 Material3 色值。
5. **国家码前缀**：`PhoneInput` 中的国家码 `Surface + border` 改为 `MenaColors.Surface` 底色 + `MenaColors.Primary` 半透明边框，与 `GoldOutlinedTextField` 视觉统一。

### 具体改造细节

#### 2.1 LoginScreen.kt

**改造 1 — 渐变背景替代 Surface**

```kotlin
// 改造前
Surface(
    modifier = modifier.fillMaxSize(),
    color = MaterialTheme.colorScheme.background
) { Column(...) }

// 改造后
Box(
    modifier = modifier
        .fillMaxSize()
        .background(
            brush = Brush.verticalGradient(
                colors = listOf(MenaColors.Background, MenaColors.Surface)
            )
        )
) { Column(...) }
```

**改造 2 — 标题文字颜色**

```kotlin
// Logo emoji — 保持不变
Text(text = "🎙️", ...)

// 主标题 "Voice Room" — 白色
Text(
    text = "Voice Room",
    color = MenaColors.OnBackground,  // 新增
    ...
)

// 副标题 "تسجيل الدخول" — 金色
Text(
    text = "تسجيل الدخول",
    color = MenaColors.Primary,  // 替代 MaterialTheme.colorScheme.onSurfaceVariant
    ...
)
```

**改造 3 — 登录按钮**

```kotlin
// 改造前
Button(onClick = onLogin, enabled = uiState.isLoginButtonEnabled, ...) {
    Text(text = "تسجيل الدخول", style = MaterialTheme.typography.labelLarge)
}

// 改造后
GoldButton(
    text = "تسجيل الدخول",
    onClick = onLogin,
    enabled = uiState.isLoginButtonEnabled,
    modifier = Modifier.fillMaxWidth().height(52.dp)
)
```
