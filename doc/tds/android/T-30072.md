## 三、TDD 验收用例
- [ ] 复用 T-30031 `ILottiePlayer`（防腐层；不直接用 com.airbnb.lottie）
- [ ] Lottie URL 由 Server 下发，客户端不硬编码
- [ ] 队列 FIFO，同时 5 个进场不会同屏叠加
- [ ] 设置关闭后不播放（但仍接收信令更新麦位徽章）
- [ ] 进场补偿事件（重连后回放）不重播：依赖 envelope.msg_id + DataStore last_played_msg_ids
- [ ] BGM 与 Lottie 时长一致（取 `duration_ms`）
- [ ] 全服跑马灯（NobleEntranceGlobal）独立队列，不与房间队列冲突
