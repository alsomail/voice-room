### 预设
```kotlin
enum class KickReason(@StringRes val label: Int, val key: String) {
    Harassment(R.string.kick_reason_harassment, "harassment"),
    Spam(R.string.kick_reason_spam, "spam"),
    Abuse(R.string.kick_reason_abuse, "abuse"),
    Other(R.string.kick_reason_other, "other")
}
```

### State
```kotlin
data class KickDialogState(
    val selected: KickReason = KickReason.Harassment,
    val customText: String = "",
    val submitting: Boolean = false
) {
    val canSubmit: Boolean
        get() = !submitting && (selected != KickReason.Other || customText.isNotBlank())
}
```

### 布局
- AlertDialog，不可外部 dismiss（`properties = DialogProperties(dismissOnClickOutside = false)`）
- Title "踢出原因"
- RadioGroup 4 项
- 选"其他"时显示 OutlinedTextField（max 100 字）
- 底部 [取消] [确认]（`GoldButton` 在 canSubmit 为 false 时置灰）

### 提交

val reasonText = when (state.selected) {
    Other -> state.customText.trim()
    else -> state.selected.key
