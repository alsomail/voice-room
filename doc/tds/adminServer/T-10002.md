# TDS: 管理员登录接口 (Task ID: T-10002)

## 一、背景与目标
- **关联需求**：product.md - Admin Server 管理后台
- **前置依赖**：T-10001 (管理员表设计)
- **本 Task 的目标**：实现 POST `/api/v1/admin/login` 管理员账号密码登录接口

## 二、方案设计
### 核心数据流
Client → POST /api/v1/admin/login {username, password} → Controller → AdminAuthService → 查 admins 表 → shared::crypto::verify_password() → 生成 Admin JWT (iss="voiceroom-admin", 含 role) → 更新 last_login_at → 写 admin_logs (action="admin_login") → 返回 {token, admin}

### 涉及文件清单
| 文件路径 | 变更类型 | 说明 |
|----------|----------|------|
| `app/adminServer/src/modules/auth/controller.rs` | 新增 | login handler |
| `app/adminServer/src/modules/auth/service.rs` | 新增 | 登录逻辑 |
| `app/adminServer/src/modules/auth/repository.rs` | 新增 | admins 表查询 |
| `app/adminServer/src/modules/auth/dto.rs` | 新增 | AdminLoginRequest, AdminLoginResponse |
| `app/shared/src/crypto/password.rs` | 引用 | bcrypt verify |
| `app/shared/src/jwt/encode.rs` | 引用 | JWT 签发 |

### 接口 / 信令定义

**前置依赖**：T-10001

**接口定义**：参见 `doc/protocol.md` §三 3.1 POST /api/v1/admin/login

**Admin JWT Claims**：
```json
{ "sub": "admin_uuid", "role": "operator", "iat": ..., "exp": ..., "iss": "voiceroom-admin" }
```
- 有效期 7 天
- iss 固定 "voiceroom-admin"（区分 C 端 JWT）

## 三、TDD 验收用例
- [ ] 正向：正确账号密码登录成功，返回 JWT + admin 信息
- [ ] 正向：JWT 有效期 7 天，包含 role 字段
- [ ] 正向：登录成功后更新 last_login_at
- [ ] 正向：登录操作写入 admin_logs
- [ ] 异常：账号不存在返回 401 (40106)
- [ ] 异常：密码错误返回 401 (40106)
- [ ] 异常：账号被禁用（is_active=false）返回 403 (40302)
