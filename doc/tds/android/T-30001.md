# TDS: 登录页 UI (Compose) (Task ID: T-30001)

## 一、背景与目标
- **关联需求**：product.md - 手机号一步登录（C 端用户）
- **本 Task 的目标**：实现 Android C 端登录页，使用 Jetpack Compose + Material3 构建，支持手机号 + 验证码登录流程，完整支持 RTL 布局（阿拉伯语从右到左）。
- **前置依赖**：T-00002 (send-code 接口)

## 二、方案设计
### 核心数据流
用户输入手机号 → 点击"发送验证码" → ViewModel 调用 send-code API → 60 秒倒计时 → 用户输入验证码 → 点击"登录"

### 关键技术决策

1. **RTL 实现**：在 `LoginScreenContent` 中通过 `CompositionLocalProvider(LocalLayoutDirection provides LayoutDirection.Rtl)` 强制全页 RTL，无需修改 AndroidManifest，支持按需切换。
2. **手机号验证**：过滤非数字后恰好 9 位（沙特标准：+966 后的 5XXXXXXXX 格式）。
3. **可测试性**：`LoginUiState` 为纯 Kotlin，无 Android 框架依赖，ViewModel 可在 JUnit4 中直接实例化。
4. **倒计时**：`viewModelScope.launch + delay(1000L)` 实现，ViewModel 销毁时自动取消。
