| 功能模块 | Yalla | YoHo | 我们的设计 | 优先级 |
|---------|-------|------|-----------|--------|
| **用户管理** | ✅ 查询/封禁/编辑 | ✅ 用户画像分析 | ✅ 查询/封禁/编辑/标签 | P0 |
| **内容审核** | ✅ 消息/房间审核 | ✅ AI+人工双审 | ✅ 待审列表+快捷操作 | P0 |
| **房间管理** | ✅ 实时监控/强关 | ✅ 房间分类管理 | ✅ 实时监控/强关/日志 | P0 |
| **财务管理** | ✅ 充值/提现审核 | ✅ 对账报表 | ✅ 订单/对账/手动充值 | P1 |
| **素材管理** | ✅ 礼物/装扮上传 | ✅ 版本管理 | ✅ 素材库+CDN管理 | P1 |
| **运营配置** | ✅ 活动配置 | ✅ 推送管理 | ✅ 活动模板+推送 | P2 |
| **数据分析** | ✅ 基础报表 | ✅ BI 大盘 | ✅ 关键指标+自定义查询 | P2 |
| **权限管理** | ✅ 角色权限 | ✅ 操作日志 | ✅ RBAC+审计日志 | P1 |

## 模块 1: 数据看板 (Dashboard) — P0

**首页展示关键指标**:
```
┌─────────────────────────────────────────────────────┐
│  今日概览                                            │
│  ┌──────────┬──────────┬──────────┬──────────┐     │
│  │ DAU      │ 新增用户  │ 活跃房间  │ 今日营收  │     │
│  │ 12,345   │ 1,234    │ 567      │ $8,765   │     │
│  └──────────┴──────────┴──────────┴──────────┘     │
│  实时在线: 3,456 人   在线房间: 234 个              │
│  [7日趋势图] [付费转化漏斗] [Top 10 主播]           │
└─────────────────────────────────────────────────────┘
```

**关键指标**: 实时在线/房间数、DAU/MAU/新增、付费转化、同比/环比

---

## 模块 2: 用户管理 — P0

1. **用户列表**: 手机号/ID/昵称搜索，状态筛选，批量操作
2. **用户详情**: 基础信息 + 资产信息 + 行为数据 + 操作按钮
3. **封禁管理**: 永久/临时封禁，原因+备注，解封流程

**差异化**: 用户标签系统、黑名单批量导入

---

## 模块 3: 房间管理 — P0

1. **实时房间列表**: 多条件筛选，按在线人数排序
2. **房间详情**: 成员列表 + 实时聊天记录 + 麦位情况
3. **强制关闭**: 记录原因，推送通知给房主

**竞品启发**: YoHo 远程静默观察；Yalla AI 敏感词标红预警

---

## 模块 4: 财务管理 — P1

1. **充值订单**: 列表/筛选/手动补单
2. **手动充值/扣减**: 补偿/测试/活动赠送，二次确认
3. **提现审核**: 主播提现 → 审核 → 打款
4. **财务报表**: 日报/月报/对账

**差异化**: 自动对账、大额异常预警

---

## 模块 5: 素材管理 — P1

1. **礼物管理**: 名称/价格/分类/特效/动画文件上传
2. **贵族配置**: 等级/价格/特权/进场特效配置
3. **房间装扮**: 皮肤/背景/音效包，版本控制

---

## 模块 6: 运营配置 — P2

1. **活动管理**: 充值返利/新人礼包/邀请有奖/主播PK
2. **推送管理**: 系统通知/运营活动/个性化推送
3. **公告管理**: App 内公告/跑马灯/版本更新提示

---

## 模块 7: 数据分析 — P2

1. **用户分析**: 新增/活跃/流失趋势，留存分析，用户画像
2. **营收分析**: 充值趋势，ARPU/ARPPU，付费漏斗
3. **内容分析**: 房间热度，礼物销量，主播收益排行
4. **自定义查询**: SQL 查询界面（仅超级管理员）

---

## 模块 8: 权限管理 (RBAC) — P1

| 角色 | 权限范围 | 典型用户 |
|------|----------|----------|
| 超级管理员 | 全部权限 | CTO、CEO |
| 运营经理 | 用户/房间/运营配置 | 运营总监 |
| 客服人员 | 用户查询/封禁/房间监控 | 客服团队 |
| 财务人员 | 充值订单/提现审核/财务报表 | 财务部 |
| 内容审核 | 内容审核/用户举报 | 审核团队 |
| 数据分析师 | 数据看板/报表导出 | 数据团队 |
| 只读 | 仅查看 | 实习生 |

**审计日志**: 所有敏感操作，日志保留 180 天
