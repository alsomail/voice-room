# TDS: Web 端活水房间监控增强 (Task ID: T-20011)

## 一、背景与目标

- **关联需求**：[doc/design/adminWeb/T-20011.md](../../design/adminWeb/T-20011.md)
- **前置依赖**：T-20004（RoomsPage + useRoomsPage + RoomsTable 已就绪）、T-20005（RoomDetailModal 已就绪）、T-10004（房间列表接口已提供 `member_count`、`status`、`created_at` 字段）
- **本 Task 的目标**：在不新增后端接口的前提下，在现有房间管理页面 `/rooms` 增加"活水监控"能力：新增活跃状态 Tag 列与持续时长列、工具栏加活跃度前端筛选、异常行高亮，帮助运营人员快速识别疑似异常房间。

## 二、方案设计

### 2.1 核心数据流

```
AdminRoomItem[]（来自 T-10004 API）
        │
        ▼
useRoomsPage（已有）
  ├─ items: AdminRoomItem[]           ← API 数据（不变）
  ├─ activityFilter: ActivityFilter   ← 新增 state（默认 'all'）
  ├─ setActivityFilter(v)             ← 新增 setter
  └─ filteredItems: AdminRoomItem[]   ← useMemo 前端过滤结果（新增）
        │
        ▼
RoomsPage（index.tsx）
  └─ <RoomsTable
       items={filteredItems}           ← 由 filteredItems 替换原 items
       activityFilter={activityFilter}
       onActivityFilterChange={setActivityFilter}
       .../>
        │
        ▼
RoomsTable（已有，扩展）
  ├─ 工具栏：新增 <Select data-testid="activity-filter">
  ├─ 新列：<RoomActivityTag> (data-testid="room-activity-tag-{roomId}")
  ├─ 新列：formatDuration(created_at) (data-testid="room-duration-{roomId}")
  └─ onRow：getActivityStatus === 'abnormal' → style 行背景高亮
```

**设计原则**：
- **活跃度筛选纯前端**：复用 T-10004 已有数据，不增加任何新 API，`filteredItems` 在 `useRoomsPage` 内以 `useMemo` 计算
- **活跃状态计算为纯函数**：抽离至 `roomUtils.ts`，便于单独单元测试，与组件解耦
- **最小化 diff**：`RoomsTable` Props 仅新增 2 个字段，`useRoomsPage` 返回值追加 3 个字段，原有接口契约不变
- **分页行为**：服务端分页的 `total` 字段不受前端活跃度筛选影响（当前页过滤，分页控件按 API 数据计算）

### 2.3 类型定义 — `roomUtils.ts`

```typescript
export type ActivityLevel = 'active' | 'quiet' | 'abnormal' | 'normal';
export type ActivityFilter = 'all' | 'active' | 'quiet' | 'abnormal';

/** 活跃状态计算（now 参数支持测试注入，保证时间确定性） */
export function getActivityStatus(
  room: Pick<AdminRoomItem, 'member_count' | 'status' | 'created_at'>,
  now: Date = new Date(),
): ActivityLevel {
  const { member_count, status, created_at } = room;
  if (member_count >= 5) return 'active';
  if (member_count === 0 && status === 'active') return 'abnormal';
  if (member_count >= 1 && member_count <= 4) {
    const durationMs = now.getTime() - new Date(created_at).getTime();
    if (durationMs > 60 * 60 * 1000) return 'quiet'; // > 1h
  }
  return 'normal';
}

/**
 * 格式化持续时长
 * < 1min  → "0m"
 * < 1h    → "35m"
 * ≥ 1h    → "2h 35m"
 * ≥ 24h   → "3d 2h"
 */
export function formatDuration(createdAt: string, now: Date = new Date()): string {
  const diffMs = now.getTime() - new Date(createdAt).getTime();
  if (diffMs < 0) return '0m';
  const totalMinutes = Math.floor(diffMs / 60000);
  const days = Math.floor(totalMinutes / 1440);
  const hours = Math.floor((totalMinutes % 1440) / 60);
  const minutes = totalMinutes % 60;
  if (days > 0) return `${days}d ${hours}h`;
  if (hours > 0) return `${hours}h ${minutes}m`;
  return `${minutes}m`;
}

/** 根据 ActivityFilter 过滤房间列表 */
export function filterByActivity(
  items: AdminRoomItem[],
  filter: ActivityFilter,
  now: Date = new Date(),
): AdminRoomItem[] {
  if (filter === 'all') return items;
  return items.filter((room) => getActivityStatus(room, now) === filter);
}
```
