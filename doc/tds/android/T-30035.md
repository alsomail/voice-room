### EventReportClient API
```kotlin
class EventReportClient @Inject constructor(
    private val queueDao: EventQueueDao,
    private val throttler: Throttler,
    private val wsTransport: WsTransport,
    private val httpTransport: HttpTransport,
    private val consent: ConsentRepository,
    private val commonProps: CommonPropsProvider,
    private val sessionManager: SessionManager,
    private val analyticsPort: AnalyticsPort     // 错误时 captureException
) {
    fun track(event: String, properties: Map<String, Any?> = emptyMap()) {
        if (consent.mode != ConsentMode.All) return  // 仅 Crash 模式直接丢弃
        val enriched = commonProps.enrich(event, properties, sessionManager.currentId)
        queueDao.insert(enriched.toEntity())
        throttler.notify(queueDao.size())
    }
    suspend fun flush() { /* 取至多 100 条 → WS/HTTP → 成功删/失败保留 */ }
}
```

### 队列策略
- Room 数据库表 `event_queue(id, event_name, properties_json, session_id, client_ts, created_at)`
- 淘汰：`size > 1000` → 删除最旧的（`LIMIT 1 ORDER BY created_at`）
- Flush 触发：
  - `size >= 8` 立即 flush
  - 或 距上次 flush >2min
  - 或 App 进后台 `onStop`
  - 或 WS 重连成功

### Transport 选择
```kotlin
suspend fun send(batch: List<Event>): Result<SendOutcome> {
    return if (wsClient.isConnected) wsTransport.send(batch)  // WS
           else httpTransport.send(batch)                      // HTTP fallback
}
```
- 成功：`queueDao.deleteByIds(outcome.acceptedIds)`
- 失败（网络错/超时）：保留，指数退避（2s,4s,8s,16s,max 60s）

### SessionManager
- 首次进前台生成 UUID 作为 session_id
- App 退后台 30s 后回前台 → 新 session_id
- 提供 `sessionManager.currentId`

### CommonPropsProvider
- 自动注入：`device_id`（安装时生成存 DataStore）、`app_version`（BuildConfig）、`os_version`、`locale`（`Locale.getDefault()`）、`network_type`（ConnectivityManager）

### 隐私合规弹窗
- `PrivacyConsentDialog` Compose：
  - Title "数据收集说明" / "جمع البيانات"
  - Body：简述会收集哪些（匿名行为 + Crash），**不含手机号**
  - 按钮：[仅 Crash] [同意全部]
- `ConsentRepository` 存 DataStore，首次 Splash 成功后 + 未设置时显示弹窗
- 选完后：`analyticsPort.setConsent(mode)` + `eventReportClient` 立即生效
