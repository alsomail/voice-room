# T-30023: 消息Tab占位页

> **归属端**: Android  
> **模块**: UI/Messages  
> **前置依赖**: T-30018 (MenaTheme), T-30020 (MainScreen)  
> **参考**: [Android App 界面设计规范 §2.5](../../product/android_app_design.md)

---

## 需求描述

IM 消息功能的占位页面，在 MainScreen 消息 Tab 下显示"即将上线"提示。为后续 Phase 1+ IM 功能预留页面结构。

## 页面布局

```
┌──────────────────────────────┐
│  消息                         │  ← TopBar: 白色标题
├──────────────────────────────┤
│                              │
│                              │
│         💬                   │  ← 金色图标 64dp
│    消息功能即将上线            │  ← OnBackgroundSecondary, bodyLarge
│    敬请期待                   │  ← OnBackgroundTertiary, bodySmall
│                              │
│                              │
└──────────────────────────────┘
```

## 通用组件提取

| 组件 | 文件 | 说明 |
|------|------|------|
| `PlaceholderScreen` | `common/ui/PlaceholderScreen.kt` | 通用占位页：icon + title + subtitle |

`PlaceholderScreen` 参数：
```kotlin
@Composable
fun PlaceholderScreen(
    icon: ImageVector,
    title: String,
    subtitle: String? = null,
    modifier: Modifier = Modifier
)
```

## 自动化测试锚点

| testTag | 元素 | 断言 |
|---------|------|------|
| `messages_screen` | 消息页容器 | Tab 切换后可见 |
| `messages_placeholder` | 占位内容区 | 包含"即将上线"文字 |

## TDD 验收标准

1. 消息 Tab 点击后显示 PlaceholderScreen
2. 显示金色图标 + "消息功能即将上线" + "敬请期待"
3. `MenaTheme` 深色背景
4. `PlaceholderScreen` 组件可在其他占位场景复用
