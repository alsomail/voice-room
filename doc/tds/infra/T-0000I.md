# TDS: package.json scripts 一键命令 (Task ID: T-0000I)

> **主设计**：[T-0000E](./T-0000E.md)

## 一、背景与目标
修复体检缺口 C9。提供 6 个 npm script 让 E2E 启动门槛降到最低。

## 二、方案设计

### 涉及文件清单
| 文件路径 | 变更类型 | 说明 |
|---|---|---|
| `package.json` | 修改 | 增加 scripts 段 |

### scripts 定义
```json
{
  "scripts": {
    "preflight": "bash scripts/dev/preflight.sh",
    "db:seed": "bash scripts/dev/seed-e2e.sql.runner.sh",
    "db:reset": "bash scripts/dev/reset-e2e.sh",
    "e2e:local": "E2E_PROFILE=local playwright test",
    "e2e:staging": "E2E_PROFILE=staging playwright test",
    "e2e:prod-smoke": "E2E_PROFILE=prod playwright test --grep @prod-safe"
  }
}
```

注意：实际命令通过 cross-env 或脚本包装确保 Windows 兼容（如团队仅 mac/linux 可直接 inline）。

## 三、TDD 验收用例
- [ ] `npm run preflight` 1 秒内完成（local 全 5 端 OK 时）
- [ ] `npm run e2e:local` 等价 `E2E_PROFILE=local npx playwright test`
- [ ] `npm run e2e:prod-smoke` 仅运行打 `@prod-safe` 标签的用例
- [ ] `package.json` 通过 `npm pkg fix` 校验
