# TDS: E2E globalSetup/globalTeardown/envLoader 三件套 (Task ID: T-0000H)

## 一、背景与目标
- 修复体检缺口 C6/C10/C12：Playwright 单文件 dotenv、缺 5 端预检、无幂等清理。
- 输出 envLoader / globalSetup / globalTeardown 三件套 + `playwright.config.ts` 联动改造。

## 二、方案设计

### 涉及文件清单
| 文件路径 | 变更类型 | 说明 |
|---|---|---|
| `tests/scripts/support/envLoader.ts` | 新增 | 单一 env 加载源；导出强类型 `E2EEnv` 接口；缺关键字段抛 `MissingEnvError` |
| `tests/scripts/support/globalSetup.ts` | 新增 | 调 envLoader → preflight → 按需 seed → 注入 `process.env` |
| `tests/scripts/support/globalTeardown.ts` | 新增 | 仅 local profile 调 reset-e2e.sh |
| `tests/scripts/support/prodSafeGuard.ts` | 新增 | profile=prod 时拦截非 GET 请求 |
| `playwright.config.ts` | 修改 | 引入 globalSetup/Teardown；从 envLoader 注入 `baseURL`；profile=prod 时设置 `grep: /@prod-safe/` |

### envLoader 类型契约（详见 T-0000E §2.7）

```typescript
export class MissingEnvError extends Error { fields: string[] }
export interface E2EEnv { /* 见主 TDS */ }
export function loadE2EEnv(): E2EEnv;
```

加载顺序：
1. `dotenv` 读取 `.env`（覆盖默认值）
2. `dotenv` 读取 `tests/scripts/env/.env.{E2E_PROFILE}`（不存在不报错，存在则覆盖）
3. 字段校验，缺则 `throw new MissingEnvError([...])`
4. 类型转换（bool/int/url）

### globalSetup 流程
```ts
async function globalSetup() {
  const env = loadE2EEnv();
  console.log(`[E2E] Profile: ${env.profile}, allowWrites: ${env.allowWrites}`);
  // preflight：local 强制全 5 项；staging/prod 仅 HTTP 3 项
  await runPreflight(env);
  if (env.profile === 'local' && env.allowWrites) {
    await execSync('bash scripts/dev/seed-e2e.sql.runner.sh', { stdio: 'inherit' });
    // 读取 scripts/dev/.seed-output.env 并 export
  }
}
```

### globalTeardown 流程
```ts
async function globalTeardown() {
  const env = loadE2EEnv();
  if (env.profile !== 'local') return; // 远端从不自动 reset
  await execSync('bash scripts/dev/reset-e2e.sh --yes', { stdio: 'inherit' });
}
```

### prod-safe 拦截（在 fixture 层）
```ts
test.beforeEach(async ({ request }, testInfo) => {
  const env = loadE2EEnv();
  if (env.profile === 'prod' && !testInfo.tags.includes('@prod-safe')) {
    test.skip(true, 'prod profile only runs @prod-safe tests');
  }
});
```

## 三、TDD 验收用例
- [ ] `E2E_PROFILE=local` + 全字段齐全 → globalSetup 成功，输出 profile 日志
- [ ] `E2E_PROFILE=staging` + 缺 `APP_SERVER_BASE_URL` → 启动期抛 `MissingEnvError`，列出缺字段
- [ ] `E2E_PROFILE=prod` 时未打 `@prod-safe` 的用例自动 skip
- [ ] `E2E_PROFILE=prod` 时 `request.post(...)` 被 `prodSafeGuard` 拦截
- [ ] globalTeardown 在 staging/prod 下不执行 reset
- [ ] envLoader 单元测试覆盖率 ≥ 90%
