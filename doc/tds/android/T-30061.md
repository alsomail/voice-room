# TDS: Billing 防腐层 (BillingPort) (Task ID: T-30061)

## 一、背景与目标
- **关联需求**：开发红线 #3 防腐层隔离 + [phase1_payment_billing.md §6.2](../../product/phase1_payment_billing.md) + [Google Play Billing 8.3.0 集成指南](https://developer.android.com/google/play/billing/integrate)。
- **目标**：定义 `IBillingPort` 接口，仅暴露领域语义；`GooglePlayBillingAdapter` 封装 BillingClient 全部 API；`FakeBillingPort` 用于单测；CI 静态检查阻止业务层直接 import `com.android.billingclient`。

## 二、方案设计

### 接口设计（领域语义）
```kotlin
interface IBillingPort {
    suspend fun connect(): BillingConnectionResult
    suspend fun queryProducts(skuIds: List<String>): List<ProductDetail>
    suspend fun launchBillingFlow(
        sku: ProductDetail,
        obfuscatedAccountId: String  // = order_id from T-00051
    ): BillingFlowResult
    val purchasesFlow: Flow<List<DomainPurchase>>  // 来自 onPurchasesUpdated
    suspend fun acknowledge(purchaseToken: String): AckResult
    suspend fun consume(purchaseToken: String): ConsumeResult  // INAPP 一次性消耗，本项目调用 ack 即可，但保留接口
    suspend fun queryPendingPurchases(): List<DomainPurchase>
    fun disconnect()
}

data class DomainPurchase(
    val purchaseToken: String,
    val productId: String,
    val obfuscatedAccountId: String?,
    val state: DomainPurchaseState, // PENDING / PURCHASED / CANCELED
    val acknowledged: Boolean,
    val purchaseTimeMillis: Long
)
```
