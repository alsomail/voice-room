# TDS: Android ChatMessageList 长按复制菜单 (Task ID: T-30053)

## 一、背景与目标
- **关联需求**：TC-CHAT-00002 Step 6 实证（tests/report-20260505-124251），长按无菜单弹出
- **本 Task 的目标**：为 UserMessageItem 增加长按 → DropdownMenu 含「复制」选项 → 写入 ClipboardManager + Toast

## 二、方案设计

### 核心数据流
```
用户长按 UserMessageItem
  → combinedClickable.onLongClick 触发
  → showMenu = true（State 变量）
  → DropdownMenu(expanded = showMenu) 弹出
  → 用户点击「复制」
  → ClipboardManager.setPrimaryClip(ClipData.newPlainText(...))
  → Toast.makeText(...).show()
  → showMenu = false（菜单收起）
```
