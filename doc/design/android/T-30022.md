# T-30022: 大厅页视觉升级

> **归属端**: Android  
> **模块**: UI/Room  
> **前置依赖**: T-30018 (MenaTheme), T-30020 (MainScreen)  
> **已有基础**: T-30005 (HallScreen) + T-30006 (Paging3) + T-30007 (CreateRoom)  
> **参考**: [Android App 界面设计规范 §2.4](../../product/android_app_design.md)

---

## 需求描述

将现有大厅页从白色主题升级为中东黑金风格，增加顶部栏和分类占位。**Paging3 分页逻辑不变**。

## RoomCard 组件升级

```
┌─────────────────────┐
│  [渐变色块/封面]       │  ← 高度 120dp，Surface色 + 随机渐变
│                     │
│  🎤 房间名称 (2行max) │  ← OnBackground, bodyMedium
│  👤 房主    🟢 87    │  ← OnBackgroundSecondary, labelSmall + OnlineCountBadge
└─────────────────────┘
整体: Surface色底, RoundedCornerShape(16.dp), padding 0dp
```

## 变更清单

| 元素 | 现状 | 升级后 |
|------|------|--------|
| 背景 | 白色 | `MenaColors.Background` |
| TopBar | 无 | 金色标题 + 搜索 + 创建按钮 |
| 分类条 | 无 | `ScrollableTabRow`（仅"热门"可用） |
| RoomCard | Material3 默认 Card | 深色 Surface + 圆角 16dp + 渐变色块 |
| 在线人数 | 纯文字 | `OnlineCountBadge`（绿点+数字） |
| 空状态 | 无 | 金色线条插画 + 提示文字 |
| FAB | BottomSheet 触发入口 | 金色圆形"+"按钮 |

## TDD 验收标准

1. 大厅页使用 `MenaTheme`，深色背景
2. RoomCard 为深色卡片 + 圆角 16dp + `OnlineCountBadge` 绿点
3. 创建房间 FAB 金色圆形，点击弹出 BottomSheet
4. **所有 Paging3 分页、下拉刷新、上拉加载逻辑不可回归**
5. 空状态显示占位图 + 提示文字
