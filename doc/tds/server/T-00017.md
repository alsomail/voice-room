# TDS: 钱包 Schema 与迁移 (Task ID: T-00017)

## 一、背景与目标
- **关联需求**：E-07 虚拟礼物与钱包闭环 MVP ([phase1_gift_economy.md §2.1](../../product/phase1_gift_economy.md))
- **本 Task 的目标**：为礼物打赏建立钻石资产与流水基座 —— `users` 表增加 `diamond_balance` 字段（CHECK >=0），新建 `wallet_transactions` 流水表，为后续送礼、管理员手动充值、WS 余额推送提供单一事实源。

## 二、方案设计

### 核心数据流
```
SendGift / AdminAdjust / (Future)Recharge
    └── BEGIN TRANSACTION
         ├── UPDATE users SET diamond_balance = diamond_balance ± N WHERE id=? AND balance ± N >= 0
         ├── INSERT INTO wallet_transactions (user_id, type, amount, balance_after, ...)
         └── COMMIT  (DB 层 CHECK 兜底负余额)
```

### 迁移脚本 (004_create_wallet.sql)
```sql
-- 1. users 增加 diamond_balance
ALTER TABLE users
    ADD COLUMN IF NOT EXISTS diamond_balance BIGINT NOT NULL DEFAULT 0
        CHECK (diamond_balance >= 0);

-- 2. wallet_transactions 流水表
CREATE TABLE IF NOT EXISTS wallet_transactions (
    id               UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id          UUID NOT NULL REFERENCES users(id),
    type             VARCHAR(32) NOT NULL, -- gift_send | gift_receive | admin_adjust | recharge | refund
    amount           BIGINT NOT NULL,      -- 正数=加，负数=扣
    balance_after    BIGINT NOT NULL CHECK (balance_after >= 0),
    ref_id           UUID,                 -- 关联 gift_record_id / admin_log_id 等
    reason           TEXT,
    operator_id      UUID REFERENCES users(id), -- 非空即管理员操作
    created_at       TIMESTAMPTZ NOT NULL DEFAULT now()
);

CREATE INDEX idx_wallet_txn_user_created ON wallet_transactions(user_id, created_at DESC);
CREATE INDEX idx_wallet_txn_type ON wallet_transactions(type, created_at DESC);

-- 3. 幂等：迁移可重入（使用 IF NOT EXISTS + DEFAULT 0）
```

### WalletTxnType 枚举（共享 crate）
```rust
// app/shared/src/models/wallet.rs
#[derive(Debug, Clone, Serialize, Deserialize, sqlx::Type)]
#[sqlx(type_name = "VARCHAR", rename_all = "snake_case")]
pub enum WalletTxnType {
    GiftSend,      // 送礼扣减
    GiftReceive,   // 收礼入账（MVP：接收者不直接加钻石，仅累加魅力；保留字段备用）
    AdminAdjust,   // Admin 手动调整
    Recharge,      // 充值（E-08 接入）
    Refund,        // 退款
}

#[derive(Debug, sqlx::FromRow)]
pub struct WalletTransactionModel {
    pub id: Uuid,
    pub user_id: Uuid,
    pub r#type: WalletTxnType,
    pub amount: i64,
    pub balance_after: i64,
    pub ref_id: Option<Uuid>,
    pub reason: Option<String>,
    pub operator_id: Option<Uuid>,
    pub created_at: DateTime<Utc>,
}
```
