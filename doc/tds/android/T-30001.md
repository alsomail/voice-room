# TDS: 登录页 UI (Compose) (Task ID: T-30001)

## 一、背景与目标
- **关联需求**：product.md - 手机号一步登录（C 端用户）
- **本 Task 的目标**：实现 Android C 端登录页，使用 Jetpack Compose + Material3 构建，支持手机号 + 验证码登录流程，完整支持 RTL 布局（阿拉伯语从右到左）。
- **前置依赖**：T-00002 (send-code 接口)

## 二、方案设计
### 核心数据流
用户输入手机号 → 点击"发送验证码" → ViewModel 调用 send-code API → 60 秒倒计时 → 用户输入验证码 → 点击"登录"

### 涉及文件清单
| 文件路径 | 变更类型 | 说明 |
|----------|----------|------|
| `app/android/app/src/main/java/com/voiceroom/feature/auth/LoginScreen.kt` | 新增 | Compose 登录页 |
| `app/android/app/src/main/java/com/voiceroom/feature/auth/components/PhoneInput.kt` | 新增 | 手机号输入组件（+966 前缀） |
| `app/android/app/src/main/java/com/voiceroom/feature/auth/components/CodeInput.kt` | 新增 | 验证码输入组件 |
| `app/android/app/src/main/java/com/voiceroom/feature/auth/components/CountdownButton.kt` | 新增 | 发送验证码倒计时按钮 |

### 接口 / 信令定义
无新增接口，本 Task 仅涉及 UI 层。ViewModel 交互由 T-30002 负责。

### UI 设计要点
- Material3 主题
- 手机号输入：国家码选择器（默认 +966 沙特）+ 号码输入框
- 验证码：6 位数字输入框
- 发送验证码按钮：倒计时 60 秒，禁用状态
- RTL 布局支持（阿拉伯语从右到左）
- 品牌 Logo 在顶部
